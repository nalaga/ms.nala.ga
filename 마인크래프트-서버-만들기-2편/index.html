마인크래프트 서버 만들기 프로젝트! #2 기초 서버 열기



*본 글은 나눔고딕과 PC버전에 최적화 되있으니 맞춰주시고 보시는것을 추천 합니다!

*나눔고딕 설치 링크: http://hangeul.naver.com/2016/nanum


안녕하세요! 도치입니다! 자 이제 드디어 서버를 열어볼 시간이 왔습니다! (와아앙)


자 이제 본격적으로 시작해볼까요?


일단 서버를 열기전에 서버를 열수 있는 버킷을 다운로드 받아야 합니다!


모두 http://tcpr.ca 이사이트로 접속해주세요!


자 그럼 이렇게 사이트가 뜰텐데 맨오른쪽에 Downloads 버튼이 있을텐데 거길 눌러주세요!


자 여기서 별표 친데를 봐주세요! 저기 써있는게 버킷 종류인데 자기가 필요하는 버킷에 따라 클릭해주세요!


버킷 종류 확인 게시글: http://doch12.tistory.com/6


자 원하시는걸 선택하셨으면 이렇게 뜰겁니다. (저 같은 경우 spigot를 선택한 화면입니다.)

자 그때 여기서 밑줄이 쳐져 있는 Uncheck All 을 눌러주세요! 그럼 다 체크해제 가 될텐데 그때 원하는 버전을 체


크 하시면 됩니다.

자 제가 버튼을 누르고 제가 원하는버전인 1.12버전을 체크 했습니다. 자 여기서 가장 위에것을 다운로

드 받아주시면 되는데 여기서 유의 하실게 있습니다! 파일 이름에 api나 shaded가 들어가 있는것은 다운받으면 안됩니다.

그건 버킷파일이 아니라 개발자용 파일입니다. 다운 받아봤자 실행 안됩니다.

자 그럼 선택 하셨으면 Download 버튼을 눌러주셔서 다운을 받아주세요.

자 그다음에 바탕화면이든 어디든 폴더 하나를 만들어 줍니다.

자 이렇게 만들어 주셨으면 아까 다운 받았던 버킷 파일을 복사해서 넣어주세요.


이렇게 복사 하셨으면 메모장을 열어주세요.


자 이렇게 적어주세요. 이건 서버 실행기를 만드는 겁니다. 버킷파일을 실행시킬 파일이에요.

저 title은 실행기를 실행시킬때 윈도우 창에 뜰 이름입니다. 서버랑은 아무상관없어요. 아무거나 적어주
시면 됩니다.
그다음에 저 Xmx 숫자G는 램 숫자를 적어주시면 됩니다. 나는 1기가를 서버에 쓰고 싶으시면 1를, 6기가를 쓰고 싶으시면 6을 써주시면 됩니다.

저 램 크기는 서버를 돌릴때 아주 중요합니다. 저 램 크기에 따라 서버 렉이 걸릴지 안걸릴지 정해줍니다.

렉 안걸리고 싶으시면 램을 많이 설정 해주시면 됩니다. 저걸 크게 쓴다고 다 할당 되지는 않습니다.

저 서버파일.jar은 서버파일 이름을 그대로 복사 & 붙여넣기 하셔서 쓰세요. 파일이름이 1이라고 틀리면

작동 안됩니다.

자 이걸 또 못쓰실 분들이 계실것 같아서 텍스트 파일로 올려드립니다.

start.txt


자 다 쓰셨으면 start.bat 로 저장해주세요. 무조건 확장자를 .txt아니고 .bat로 쓰셔야 합니다.

자 저장까지 하셨으면 아까만든 서버 실행기를 클릭해서 실행해주세요.


자 근데 이렇게 뜨면서 멈춥니다. 이게 맨처음 시작 할때만 뜨는건데요. 바로 율라, 즉 마인크래프트 라이

선스를 지켜야 된다는 겁니다. 이제 다시 폴더를 보시면 파일들이 막 생성이 되있을텐데 그중에서

eula.txt 파일을 실행시켜 주세요.


자 그럼 이렇게 뜹니다. 여기서 밑줄이 쳐져있는 false를 true로 바꿔주세요. 그다음에 다시 실행 시켜주시면!


짜잔~ 이렇게 완료가 됩니다! 자 이제 한번 들어가 볼까요?

맞는버전으로 들어가 서버 주소에 localhost 라고 치고 들어가 보세요! 과연…?


짜잔~ 이렇게 접속이 되는걸 확인하실수 있습니다! 이렇게 해서 기본 서버여는 법을 다 배우셨습니다!

짝짝짝 자 이렇게 여셔서 친구들이랑 같이 하시면 되고, 더 많은 기능 원하시는 분들은 저의 다음 게시글

을 봐주시면 될것 같습니다. 다음 게시글은 서버 설정방법 및 서버 기초 명령어 입니다!

많은 관심 부탁드립니다~ 안녕~
