그럼 API 를 통해서 콘텐츠를 업로드 하는 방법입니다.

 

API 를 이용하는 어플리케이션, 앱, 웹에서 손쉽게 콘텐츠를 V CASE 로 업로드 할 수 있습니다.

 

 

 

콘텐츠를 업로드 하기 위한 API 호출에 대한 인터페이스 명세서 입니다.

http://upload.vcase2.myskcdn.com:8081 의 URL에 호출하여야 하며, API 관리에서 발급 받은 access_token 값과 업로드 할 영상 콘텐츠의 바이너리파일을 file 객체에 담아 전달하는 것이 필수 파라메터로 되어 있습니다.

그 외에는 필수 값이 아닌 옵션입니다. tags 파라메터를 입력하면, 영상에 태그를 입력할 수 있습니다. playlist_ids 파라메터에 입력된 값은 플레이리스트 식별 아이디 리스트로 각 아이디는 콤마(,)로 구분할 수 있습니다.

파일 업로드 하는 간단한 샘플을 작성하였습니다.

 

POST 방식으로 http://upload.vcase2.myskcdn.com:8081 의 URL 에 호출하며, from 태그에

 

multipart/form-data 방식으로 전달하게 속성을 지정하였습니다.

 

파일 객체에 파일을 담다 file 파라메터로 전달하고, 인증을 위해서 accsee token 에 값을 담아 전달하였습니다.

 

 

 

 

파일선택을 눌러 업로드할 파일을 선택하여 줍니다.

 

단 jQuery 업로드의 제약으로 5MB 이하의 영상을 선택하여야 합니다.

 

SEND 를 눌러 function 을 호출합니다.

 

 

 

 

파일이 성공적으로 업로드 되었다고 alert 알림창이 뜨게 됩니다.

 

 

 

 

V CASE 의 콘텐츠에 API 로 업로드 요청한 콘텐츠가 업로드 된 것을 확인할 수 있습니다.

 

API 제공되는 범위는 크게 콘텐츠 업로드&관리, 플레이어 관리, 라이브 채널 관리, 라이브 DVR 프로파일러, 편성, 퍼블리싱 관리, 광고 콘텐츠 업로드/관리, 통계, 방송관리, 채팅관리, 방송 지역 광고 관리, 서비스 인증 토큰 관리, 코드 관리로 분류됩니다.

 

콘텐츠 관련된 상세 관리는 콘텐츠 파일 업로드, 재업로드, 원격 콘텐츠 업로드, 콘텐츠 메타 정보수정, 활성화 상태 수정, 콘텐츠 내 자막 바인딩/언바인딩/콘텐츠 목록 조회, 단일 조회, 콘텐츠 썸네일 제출, 콘텐츠 자르기/붙이기 등이 있어 API 를 통한 상세 설정이 가능합니다.

 

 

 

 

 

업로드 알람 설정은 대용량 미디어 파일을 효과적으로 업로드 관리하는 방법으로 CallBack 을 받아

고객사 서버로 상태정보를 전달하여 줍니다.

 

업로드가 정상적으로 되었는지 성공, 실패 등의 정보를 담아 줄 수 있습니다.

 

 

 

rev 파라메터를 추가로 입력하여 API 를 호출하게 되며, 업로드 시 입력한 사용자 매개변수를 그대로 돌려 줍니다. 그로 인해 고객사가 관리하는 데이타베이스(DB)에 로그 기록 및 추가적인 정보를 기록할 수 있게 합니다.

 

 

 

 

전송작업이 완료되면, 성공/실패에 해당되는 내용을 설정한 URL 로 알람 메시지를 전달해 줍니다.

 

콜백 URL 은 GET 또한 POST 메소드를 지정하여 원하는 방식으로 받을 수 있습니다.

 

http://user-service-domain.com/vcase-transmission-callback?contents_id={content_id}&state={state}&job_publish_key={job_publish_key}&target_co ntent_id={target_content_id}&rev={rev}

 

user-service-domain.com/vcase-transmission-callback 은 고객사 서비스에서 직업 상태에 대한 콜백을

 받을 수 있도록 제작한 URI 입니다. 

 

{contents_id}, {state}, {rev}와 같은 Vcase 의 시스템 메타 정보이며,

상기 URI에서 지정한 방식대로 전달하게 됩니다.

 

 

 

어플리케이션, 웹, 앱까지 다양한 범위에서 사용할 수 있는 V CASE 에서 제공되는

SDK 및 API 연동에 대해서 알아 보았습니다.

 

인증 토큰을 통해서 V CASE 의 기능을 모두 API 로 제어할 수 있는 강력함을 제공하여 주었습니다.

 

고객사가 원하는 개발 시 필요한 모든 API 제공되며, 손쉬운 온라인 비디오 플랫폼을 사용할 수 있게

해 주었고, 손쉬운 연동으로 개발의 시간적인 단축을 가능하게 해줍니다.
