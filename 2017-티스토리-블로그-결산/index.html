2017 티스토리 블로그 결산

  • By admin
  • on 1월 12, 2018
  • in 미분류

3줄 요약

  1. 티스토리에서 블로그 결산을 깔끔하게 해줘서 포스팅 해봅니다. 500명 선정해서 선물도 준다는데, 됐으면 좋겠네요.
  2. + IT인터넷 : 주로 포스팅하는글이 웹 개발에 관련된 글입니다.+ 친절 …중략… 이시더군요. 상위에서 1%의 차이는 정말 엄청난 차이인것 같습니다.^^
  3. 제 블로그를 방문해주시는 모든분께 새해 복 많이 받으시길 바랄께요.^^

이제 블로그를 시작한지 1년차가 됩니다. 자꾸 다시 찾아보게 되는 내용들을 정리해두겠다는 생각으로 시작을 했는데, 조금씩 욕심을 내다보니 구글 애널리틱스, 애드센스도 붙이게 된 블로그 입니다.

IT 관련 글들을 적어서 그런지, 주 방문 검색엔진이 구글 입니다. 방문하시는 분들의 주 사용 브라우저는 구글 크롬이구요. 네이버나 다음에서 들어오는 트래픽은 거의 없는것 같습니다. 네이버쪽 방문 비중을 늘려보려고 이것저것 해봤지만,  전혀 늘지 않았습니다. 이제는 그런가 보다 하고 신경 끄고 있습니다.

티스토리에서 블로그 결산을 깔끔하게 해줘서 포스팅 해봅니다. 500명 선정해서 선물도 준다는데, 됐으면 좋겠네요.

1. 여섯개의 뱃지를 받았습니다.

+ IT인터넷 : 주로 포스팅하는글이 웹 개발에 관련된 글입니다.

+ 친절한댓글러 : 댓글 달아주시면 답글은 꼭 하는 편입니다. 사실 댓글이 그렇게 많지 않습니다.^^

+ 20만+ 방문자 : 예상외로 많이 방문 해주셨습니다. 저의 블로그를 방문하신 모든분께 다시한번 감사드립니다.^^

+ 200+ 포스팅 : 1일 1포스팅을 하겠다고 마음 먹었는데 쉽지 않았습니다. 1일 하나 이상을 포스팅하시는 분들도 많던데, 존경스럽습니다.

+ 1년차 블로그 : 이제 1년차 입니다. 더욱 노력해야 겠습니다.

+ 상위 5%부지러너 : 제가 자주 들러는 년간 백만정도의 방문자가 찾는 블러거분이 상위 3% 부지러너 이시더군요. 상위에서 1%의 차이는 정말 엄청난 차이인것 같습니다.^^

2. 1년간 252개의 글을 작성했습니다. 1일 1포스팅의 계획은 쉽지 않은 것이었습니다.

올해는 좀 느긋하게 2일에 1 포스팅 정도를 계획할 생각입니다. 그럼 1년에 180개 정도의 글이 포스팅 될것 같습니다.





3. 한해 동안 작성한 글중에 많이 나온 단어의 구름태그 입니다.

사용, 파일, 설치, 데이터, 실행, 설정, 방법 등의 단어가 많이 사용되었습니다. 기계적인 분석이라 그런지 별 의미가 있어 보이지는 않습니다. 프로그램의 설치 관련 글들이  인기가 있었던것은 사실인것 같습니다.

4. 주제별 발행한 글의 갯수 입니다. IT 인터넷 분류를 글을 주로 작성했습니다. 일상 글도 좀 써보려고 했는데, 블로그의 주제와 벗어나는 내용의 글을 같이 적어도 될지 고민하게 되었습니다. 결론은 많이는 안적는 것으로 정했습니다.

5. 2017년 한해 동안 244,214분이 방문해 주셨습니다. 방문하셨던 모든 분께서 도움되는 정보를 얻어셨다면 좋겠습니다.

6. 가장 많이 읽은 글은 안드로이드 스튜디오 설치하기로 15,939명이 읽었습니다. 안드로이드 프로그램에 대한 인기가 높은것 같습니다. 웹 프로그래밍을 주로 하고 있지만, 안드로이드도 배워보고 싶어서 시작을 했는데, 진도가 잘 안나가 는것 같습니다. 새로운것을 배우기는 쉽지 않은것 같습니다.

7. 한 포스트가 평균 1,172번 조회 되었습니다. 블로그에는 글 당 조회수를 알 수가 없는데, 내부적으로는 데이터가 다 있는 모양 입니다. 블로그 사용자도 이 정보를 알 수 있었으면 좋겠네요. 사실 글을 써 놓고도 전혀 조회되지 않는 글들도 분명 있을 겁니다.

8. 가장 많은 공감을 받은 글입니다. 하반기에 작성할 글로 기억이되는데, JDK9 가 나오고, Eclipse도 새버전 Oxygen 이 나온후 이 최신 버전들의 조합에 문제가 있었던 것입니다. 예상외로 많은 분들이 최신 버전을 설치하고 사용하시는 것 같았습니다.

9. 182번의 공감을 받았습니다. 인기 있는 블로그의 글 하나의 공감에 도 미치지 못하는 숫자 이군요. 더욱 노력해야 겠습니다.

10. 가장 많은 댓글이 달린 글 입니다. 사실 한분과의 대화식 댓글로 많은 분들이 댓글을 달아 주신것은 아닙니다. 소통할 수 있는 블로그가 되도록 더욱 노력하여야 겠습니다.

11. 총 89개의 댓글이 달렸습니다. 반을 내가 단 답글 일테니 실제 댓글은 40여개 정도 되겠습니다. 프로그래밍 관련 블로그이다보니 댓글의 대부분 문의 관련 글이 많습니다. 일을 하고 있어서 바로 바로 답글을 달지는 못하지만, 아는 한도내에서 꼭 댓글을 달아 드리고 있습니다. 글을 보시고 궁금한 점이 있을때는 댓글로 문의하시면 최대한 답변을 드리겠습니다.

2017년 스스로 평가해 볼 때 제법 열심히 블로그를 운영해온 것 같습니다. 처음 세운 계획대로 되지는 못했지만,

어느정도 만족할만한 성과를 낸것 같습니다. 2018년은 좀 더 마음을 가라 앉히고, 2일에 1포스팅 정도의 분량으로 좀 더 퀄리티 있는 글을 작성하는데 노력해야 겠다는 다짐을 해 봅니다.

제 블로그를 방문해주시는 모든분께 새해 복 많이 받으시길 바랄께요.^^
