3줄 요약

  1. 이즈미 : !! (저 아이, 낮에, 극단을 찾아다니던 아이야! 게다가 분명, 단원 기숙사에 대한걸 물어봤어!)
  2. 이즈미 : 신입 단원 후보.
  3. 이즈미 : 남이 듣기에 이상한말 하지마!

[마지막 찬스]

??? : ……후우. 좀처럼 조건에 맞는곳은 없네. 

이즈미 : !! (저 아이, 낮에, 극단을 찾아다니던 아이야! 게다가 분명, 단원 기숙사에 대한걸 물어봤어!)

저기! 단원 기숙사를 갖춘 극단에 관심없니!?

??? : 네? 어라, 낮에…….

이즈미 : 그때, 극단을 찾고있다고 말했었지? 벌써 찾았니?

??? : 아뇨,아직……단원 기숙사가 있는 극단이 있나요?

지배인 : MANKAI 컴퍼니,  MANKAI 컴퍼니를 잘 부탁드립니ㅡ다!

단원 기숙사 완비, 매일 두 끼 식사를 제공하는 MANKAI 컴퍼니라고 합니다!

??? : 매일 두 끼씩 제공……?

이즈미 : 그래! 어떠려나?

마스미 : 이녀석, 누구

이즈미 : 신입 단원 후보.

마스미 : 내가 있는데도?

이즈미 : 아직 부족해.

마스미 : 당신, 누구라도 좋은거군……가볍네

이즈미 : 남이 듣기에 이상한말 하지마!

마스미 : 내가 있으면 되잖아?

이즈미 : 잠깐 다물고 있어, 마스미군.

이 애는 신경쓰지 않아도 괜찮으니까, 입단을 생각해보지 않을래?

??? : 저, 전혀 경험없는데도 괜찮나요?

사쿠야 : 저도 미경험이지만 입단했습니다. 같이 열심히 해봐요!

지배인 : 저도 연극을 시작한지 8년이지만, 지금도 미경험 동지입니다!

이즈미 : (그건 어떨지 모르겠지만……)

??? : 그리고, 저, 배우라고 하기보다, 극작가 희망이라 각본을 써보고 싶은데…….

지배인 : 마침 현재 결원이 있으므로 문제 없습니다!

이즈미 : (그건 틀림없이 지금의 각본보다 나쁘게 될리는 없겠지……)

그러면, 극작가 겸 배우인걸로, 어때?

??? : 알겠습니다. 잘 부탁드립니다!

지배인 : 됐다ㅡ!

??? : 저, 미나기 츠즈루라고 합니다.

이즈미 : 잘 부탁해, 츠즈루군!

사쿠야 : 앞으로, 잘 부탁드립니다!

마스미 : 나라도, 각본정도는…….

이즈미 : 경쟁하지 않아도 좋으니까!

벌써 해가 저물어버려. 서둘러 극장에 돌아가야해!

츠즈루 : 네? 뛰는건가요?

이즈미 : 사정은 뛰어가면서 설명할테니까! 지금은 일단 뛰어!

츠즈루 : 뭔가 이유가 있는듯한데…….

마스미 : 알겠어. 당신을 따라갈께.

사쿠야 : 서두르죠!

지배인 : 헥,헥,헥…….

이즈미 : 지배인, 뛰기 시작한지 얼마나 되었다고 그래요!

후루이치 : 호오, 도망치지는 않은건가.

사쿠야 : 하아,하아,……어쨌든 시간에는 맞췄,지요!?

지배인 : 허억 허억, 신……허억 허억, 입단……허억 허억.

츠즈루 : 무리하지 않는편이 좋아요, 지배인.

이즈미 : 하아,하아, 신입단원, 데리고왔습니다!

마스미 : ……왔어.

후루이치 : 모가지는 날아가지 않은건가……운이 좋은 녀석들이군.

지배인 : 그,그럼, 철거는 이걸로……!?

이즈미 : 다행이다…….

후루이치 : 뭘 안심하고 있는건가. 나는 단원이 모아지면, 오늘은 이것으로 퇴각한다고 말한 것 뿐이다.

낮에 설명한 것처럼, 신입단원이 두 사람 늘었다는것으로, 이 극단이 존속된다고는 하지 않았다.

지배인 : 네에에!

이즈미 : 그런…….

마스미 : 벌써 해산?

츠즈루 : 사정이 너무 가혹하잖아.

지배인 : 기껏 신생 봄조가 탄생할 수 있었는데……. 역시 소용없었어……줬다 뺐다니 잔인해…….

하루이치 : 신생봄조?

지배인 : 그래요! 좀 더 모이면 신생봄조의 공연이ㅡㅡ!

하루이치 : 그 신생봄조라는 녀석들의 개막 공연은 언제 하는 건가.

지배인 : 네? 음, 언제쯤일까요, 감독?

이즈미 : 네!? 저? ……으ㅡ음, 만약 개막 공연을 한다고 치면, 천천히 기초부터 연습을 해서 반년 후 정도는…….

후루이치 : 그렇게 기다릴리가.

이즈미 : 그래도, 모두들 경험도 적고 이제 막 모인거에요. 그정도의 시간은 필요해요.

하루이치 : 다음달이다.

이즈미 : 네?

하루이치 : 죽을 정도로 맹연습해서, 다음달 중에는 공연을 해라.

이즈미 : 다음달이라니, 한달밖에 남지 않았다는 거죠!? 무리에요!

지배인 : 귀신! 악마! 서서히 괴롭히려 하다니,도S 캐릭터로 인기를 얻으려는 건가요!?

하루이치 : 의미 모를 말은 하지마라. 특별히 마지막 찬스를 주려고 하고 있다.

지금부터 말하는 세 가지 조건을 충족시킨다면, 빚의 변제는 잠시 기다리지 못할 것도 없지.

이즈미 : 조건……?

(이 때는, 몰랐다ㅡㅡ 앞으로 기다리고 있는 위기나 곤란.

수많은 만남과 감동)
