네이버페이 포인트 인출 내 계좌로 송금받기!

  • By admin
  • on 1월 15, 2018
  • in 미분류

3줄 요약

  1. 2. 클릭한 후 전환된 화면을 보니 취소, 인출 가능한 포인트가 27,0 …중략… 환불이 불가능하므로 해당 부분이 ” 0 ” 으로 나타날 수 있습니다. )
  2. 3. 클릭 후 팝업창으로 위와 같은 화면이 표기되고 내용을 살펴보면 신청 …중략… 출계좌를 지정해두지 않으신분은 같은 화면에서 은행을 설정할 수 있습니다.
  3. 5. 환불신청하기를 누르자 마자 휴대전화, 이메일, 네이버 알림 3곳에서 환불신청에 대한 메시지가 왔고 내용은 위와 같았습니다.

안녕하세요. 꿀팁킹입니다. 지난달 네이버페이 충전 이벤트 소개드린적 있었는데 다들 포인트는 많이 받으셨나용? 저도 소소하지만 네이버페이로부터 기분 좋은 선물을 받았습니다. 기대했던 9만 포인트는 아니었지만 참여상 6,000 포인트! 네이버 감사해요!

원래 포인트 받으면 적립포인트와 충전포인트로 배달의 민족 결제하면서 치킨 할인받아 먹으려 했는데 김생민의 영수증을 애청해서 그런지 요즘 배달외식이 좀 줄었습니다. 그러다가 일단 충전금액 환불받고 주문할 때 재충전 해야겠단 생각으로 환불신청을 해보았습니다. 

 

1. 먼저 네이버페이 홈페이지에 로그인 한 후 ” 포인트 ” 를 클릭! 그리고 충전포인트를 클릭해줍니다. 오른쪽 큰 부분도 있고 왼쪽의 작은 부분도 있는데 어떤 부분을 클릭해도 됩니다. 전 왼쪽 클릭함! ㅎ

2. 클릭한 후 전환된 화면을 보니 취소, 인출 가능한 포인트가 27,000 보입니다.  취소하기를 클릭해 보았습니다. (직접 충전하지 않았거나 이벤트로 받은 포인트는 환불이 불가능하므로 해당 부분이 ” 0 ” 으로 나타날 수 있습니다. ) 

3. 클릭 후 팝업창으로 위와 같은 화면이 표기되고 내용을 살펴보면 신청금액과 이메일주소 인출계좌번호를 확인할 수 있습니다. 인출계좌를 지정해두지 않으신분은 같은 화면에서 은행을 설정할 수 있습니다.



4. 충전했던 내역들이 위와 같은 화면으로 표기되고 하단으로 드래그 해보면 “신청하기” 버튼이 있습니다. 여기서 아쉬운 점은 부분취소가 어려운 점이었습니다. 1만원 정도는 남겨두고 치킨 할인받고 먹으려 했는데 제가 방법을 못 찾은 것인지 찾을 수가 없어 전액 환불로 진행하였습니다.


5. 환불신청하기를 누르자 마자 휴대전화, 이메일, 네이버 알림 3곳에서 환불신청에 대한 메시지가 왔고 내용은 위와 같았습니다.


결론 : 네이버페이 포인트 인출 안된다! 된다! 갑론을박 많았던 것으로 압니다. 하지만 직접 해보니 됩니다. 네이버페이 충전을 꺼려하시는분들~~~~ 됩니다! 충전 너무 걱정 마세요. 그리고 충전방법이 여러가지 형태가 있기 때문에 취소도 여러가지 형태로 취소가 되네요. 현금이냐, 카드냐, 신용카드냐, 기프트카드냐, PayPal이냐 등등등… 자세한 내용은 위 환불규정을 참조하여 본인에게 맞는 방법을 참고하시면 되겠습니다.
