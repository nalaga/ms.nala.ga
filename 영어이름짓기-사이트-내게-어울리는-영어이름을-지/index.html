영어이름짓기 사이트! 내게 어울리는 영어이름을 지어보자

  • By admin
  • on 1월 13, 2018
  • in 미분류

3줄 요약

  1. 그런 의미에서 여러분들도 지금 제 포스팅을 보고계시리라 생각합니다. ლ( ╹ ◡ ╹ ლ)
  2. 한국부터 미국, 캐나다, 브라질 등 다양한 국가의 이름 수뉘를 검색할 수 있슴돠
  3. 모던한 이름. 클래식.. 그리고 스마트한 이름.. 귀족 이름등 ..

한국에 살면서.. 영어이름의 필요성을 잘 느끼지 못하지만.

해외유학.. 혹은 어학연수를 갈 경우엔 꼭 영어이름짓기를 고민하게 됩니다.

 

미국에서 내 한국이름을 그대로 써도 되나?

아니면 간지나는 영어이름을 하나 만들어야할까?..

그런 의미에서 여러분들도 지금 제 포스팅을 보고계시리라 생각합니다. ლ( ╹ ◡ ╹ ლ)

 

제가 영어이름짓기 추천 사이트를 하나 소개해드리려고 해요~

바로 고우해커스 사이트의 영어이름짓기 컨텐츠인데요.. 꽤 아기자기한 모습입니다.

 

 

 

 

저도.. 제 이름은 부모님이 지어주신 저만의 Identity라고 생각하기에

처음에는 따로 영어이름을 짓지 않았었습니다.

 

그치만.. 한국이름이 발음하기 어려운지ㅠ 미국에서 교환학생을 하면서

미국 친구들이 제 이름을 못부르더군요……. 이것도 은근 스트레수ㅠㅠ

 

 

그래서 결국… 한국이름의 이니셜을 따서 영어이름을 만들었어요 🙂

 

저처럼 이니셜을 바탕으로 영어이름을 만드실것이라면..

고우해커스 영어이름짓기 컨텐츠에서 아래처럼 이니셜 영어이름찾기를 클릭하시면 되요!

 

 

 

 

자.. 그럼 지금부터 포스팅용으로 이름 하나 검색해봤습니다. ㅋㅋ

어제 너의 목소리가 들려를 못봐서….

슬픈 마음에 박수하 [P S H] 영어이름을 울면서 찾습니다.ㅠㅠ

 

 

 

박수하 이니셜로 영어이름이 참 많이도 나오네요….

저는 무엇보다 필라소피가 맘에드네여 ㅋㅋ 아주 철학적인 영어이름인듯..

박수하….. 스테파노도 영어이름으로 괜찮은듯해요ㅎㅎㅎ

 

Adolphus / Alphonse / Alphonsus / Bishop / Christopher / Daphnis / Hesperos / Joseph / Kristopher

/ Morpheus / Orpheus / Palash / Paramesh / Parish / Parthenios / Pasha / Pashenka / Pesah

/ Phelps / Philosophy / Phineas / Phinnaeus / Phipps / Phoebus / Pinchas / Porthos / 

Seraphim / Shakespeare / Shep / Shepherd / Sisyphus / Stephanos / Stephanus / Stephen

/ Theophilos / Theophilus / Thompson

 

 

아.. 그 밖에도 나라별로 이름을 찾는 것도 있어요~

한국부터 미국, 캐나다, 브라질 등 다양한 국가의 이름 수뉘를 검색할 수 있슴돠

 

 

 

한번 살펴볼께요~

한국에서 가장 인기있는 남자이름은 민준이와 지후..

그리고 서연이와 민서인가봐요 생각보다 내 주변에 없는데..

 

 

미국은 제임스와 좐이 차지했네요 ㅋㅋㅋㅋ

여자는 제니풔.. 주변에 많아요 저희 기숙사에도 한명 있음 ㅋㅋㅋㅋ

 

 

추가적으로 테마별로 영어이름 찾는 것도 있으니

모던한 이름. 클래식.. 그리고 스마트한 이름.. 귀족 이름등 ..

한번 찾아보시길 추천드릴께여 ㅋㅋ

 

 

도움이 되셧기를 바라며.. 이만 포스팅을 마치겠습니다. ^^
