롤 시즌8 말자하 템트리및 룬 스킬트리 알아봐요

  • By admin
  • on 1월 16, 2018
  • in 미분류

1줄 요약

  1. 일단은 봉인풀린 주문서로 통해 스펠을 변경이 되기 때문에 순간이동을 사용 …중략… 고 순간이동으로 복귀를 해주셔도 되는 부분 되겠습니다 제일 무난하겠습니다

안녕하세요 오늘 롤 시즌8 말자하 템트리및 룬에 대해 알려 드릴려고 합니다 말자하

의 경우에는 라인 장악력이 매우좋은 챔피언이라고 생각이 되는 부분이 되겠습니다 

라인클리어도 매우빠른편에 속하는 챔피언 되겠으며 상대방 한명을 제압하는 스킬

을 보유하고 있게 되겠습니다 전투시 궁극기 하나만으로 엄청난 모습을 보여줄수가

있다고 생각 되겠습니다 아이템 나오면 나올수록 엄청나게 강력하다고 생각되네요

1.말자하의 경우에는 침묵 + 암살자 + 제압 + 지속딜이 강력한 챔피언이 되겠습니다

일단은 패시브로 통해서 스킬을 한번 막아줄수 있고 공격을 막아주실수 있겠습니다

일단은 패시브부터 엄청나게 좋은 챔피언이라 생각 되겠습니다 저의 경우에는 말자

하가 나오면 까다롭다고 생각 됩니다 딜러를 할경우 수은 장식띠 외 안풀리기 때문

에 상대하기에 힘들다고 생각 됩니다 라인도 빨리미는 챔피언이라 까다롭겠습니다

2.룬의 경우에는 영감 + 마법을 사용을 해주시면 되겠습니다 콩콩이 소환도 좋습니다

봉인 풀린 주문서 : 소환사 파편으로 통해서 게임중 상점에서 스펠변경이 되겠습니다

또한 스펠의 쿨타임 감소 해주기 때문에 말자하에게 매우잘어울리는 룬 되겠습니다

저는 이룬을 블라디미르를 할때에 사용을 하고 말자하를 플레이할때도 사용 합니다

완벽한 타이밍 : 초시계 획득 존야하위템이 되겠으며 600원을 아낄수가 있게 됩니다

마법의 신발 : 10분후 신발이 획득됩니다 이신발이 좋은 장점은 이동속도가 10이 더

빠르기 때문에 매우좋은 룬이라고 생각이 됩니다 일단 룬이 매우 잘어울리겠습니다

기민함의 경우 중후반에 유체화로 변경 해서 플레이하면 데미지가 많이 증가합니다

3.말자하 아이템트리에 대해서 자세하게 알아보도록 하겠습니다

모렐로노미콘 – 리안드리의 가면 – 라일라이의 수정홀 – 존야의 모래시계 – 보이드

상황 아이템 : 밴시의 장막 + 루덴의 메아리 + 영겁의 지팡이 + 심연의 가면 + 라바돈

의 죽음모자 정도로 봐주면 되겠습니다 일단은 3코어까지만 나와준다면 엄청나게 강

력한 모습을 보여줄수가 있게 되겠습니다 E스킬만 걸어도 지속딜이 매우강력한 편이

라고 생각 되겠습니다 4초동안 리안드리 + E스킬데미지 + 라일라이로 통해서 슬로우

와 추가적인 데미지가 들어가기 때문에 한타에서도 강한 모습을 보여줄수 있습니다

4.말자하 신발의 경우에는 3가지를 선택을 해주시면은 되겠습니다

아이오니아 : 스펠쿨타임 + 스킬쿨타임이 줄어들기 때문에 효율이 매우좋습니다

또한 봉인풀린 주문서와 매우잘맞기 때문에 스펠 매우빨리 돌아오게 되겠습니다

마법사의 신발 : 저는 잘안올리긴 하는데 데미지면에서는 강력한 한방을 가집니다

헤르메스 : 상대방에 AP딜러들이 엄청많거나 CC기가 강력할 경우에 올려줍니다

5.스킬선마 순서의 경우에는 E-Q-W-R 순서대로 마스터를 해주시면 되겠습니다

Q스킬 : 데미지증가 + 침묵시간이 증가하고 광역데미지가 강력한 편이 되겠습니다

한타에서 침묵으로 통해서 상대방 딜러들이 스킬을 사용 못하게 해줄수 있겠습니다

W스킬 : 공허충을 소환 하는 스킬 + 유지시간 증가 + 데미지가 증가하게 되겠습니다

E스킬 : 쿨타임 감소 + 데미지증가 되겠습니다 지속딜이 매우강력한 스킬이 됩니다

또한 미니언이나 챔피언 처치시 다른 몹에게 옮겨지기 때문에 라인전에서는 라인을

빠르게 밀어줄수가 있는 스킬 되겠습니다 스킬 선마 순서의 경우에는 고정이 됩니다

6.스펠의 경우에는 순간이동 + 점멸을 사용을 해주시면은 되겠습니다

일단은 봉인풀린 주문서로 통해 스펠을 변경이 되기 때문에 순간이동을 사용합니다

점화나 유체화나 회복이나 자유롭게 사용을 해주셔도 되는 부분이 되겠습니다 저의 

경우에는 킬각이 나오는 챔피언이라면 점화를 사용을 해서 압박을 해주는것이 좋다

고 생각이 되겠습니다 만약 스펠을 못쓰고 죽거나 집을 갈경우 텔레포트로 변경을

해주고 순간이동으로 복귀를 해주셔도 되는 부분 되겠습니다 제일 무난하겠습니다
