서브타입냉동기 화성써모

서브타입 냉동기는 별도의 디젤엔진과 왕복동콤프레샤(저온용 콤프레샤)를 이용하여 가동하는 방식으로, 2.5톤 이상 11톤의 중대형트럭에 적합한 냉동기입니다.


HT-70DW Mega-F

    모 델    

HT-70DW Mega-F : 컨덴서/에바 일체형, Front mount 타입, 디젤엔진을 이용하여 냉동기 가동

HT-70DWES Mega-F : 컨덴서/에바 일체형, Front mount 타입, 디젤엔진 또는 전기모터를 이용하여 냉동기 가동(전기모터는 선택사양임)


    Standard Features 기본 사양     

▶ 엔진(디젤)

   · 모      델 : Yanmar 3TNV-82A

   · 원  산 지 : 일본

   · 실린더수 : 3기통

   · 용      량 : 1331cc

   · 엔진오일량 : 11리터


▶  컴프레서(Open Type compressor)

   · 모      델 :  화성써모 HTC500R 

   · 실린더수 : 4기통

   · 용      량 : 500cc

   · 오 일 량  : 2.6리터


▶  Refrigeration capacity(냉동능력)  외부온도 38℃ 기준

▶ 운전석 온도조절 시스템

   · 최신형 LCD 모니터의 온도 컨트롤러

   · 설정온도에 따른 엔진의 자동정지 및 가동(수동전환 가능)

   · 시스템 모니터링 가능(Low 배터리, 쿨란트, 엔진오일 감지)

   · 편리한 냉동냉장 온도 조절(냉동냉장 자동온도 표시)

   · 자동 및 수동 엔진 시동 기능 

   · 엔진가동시간 체크 기능

▶ 냉동기 시스템 보호 장치

   · 엔진 오일 소량 시 엔진 정지 장치

   · 엔진 부동액 소량시 엔진 온도상승 방지 장치

   · 고압 차단 장치

   · 저압 차단 장치

   · 전기 모터 과부하 방지 장치

▶ 제상 시스템(성에 제거)

   · 수동제상 : 사용자가 수동으로 제상버튼을 이용하여 성에 제거

   · 타이머제상 : 컨트롤러에 제상주기 및 시간을 입력, 자동 성에 제거

   · 자동제상 : 에어스위치 이용하여 자동 성에 제거

▶ 부식 및 녹 방지를 위한 외장재 및 부품

   · FRP 전면 그릴 및 사이드 도어

   · 알미늄 재질의 에바 케이스

   · 마후라, 냉각수통, 자바라, 보강대, Accumulator, 수액기 스테인레스 사용

▶ 에바 Airflow 볼륨

   · 샤프트하우징에 의한 두개의 에바 휀(290mm 직경) 가동

   · 에어볼륨 3000cfm

▶ 냉매

   · 오존 파괴 지수 “제로”, 환경 친화적 신냉매 R-404a 사용

   · 냉매주입량 : 6.8kg

▶ 중량(approximate)

   · HT-70DW Mega-F : 580kg

   · HT-50DWES Mega-F : 640kg



   Optional Features  선택 사양   

▶ 전기모터

   · 엔진을 정지하고, 전기를 이용하여 냉동기 가동(HT-70DWES Mega-F)

   · AC 모터 : 220Vac / 60Hz 삼상, 12마력, 기타 산업용 볼트(380/415V) 가능


▶ 하부 장착 컨덴서 HT-70DWU Mega-F  : 컨덴서를 차량 하부에 장착하는 분리형 타입

▶ 냉동냉장 : 두개 이상의 적재공간에 두개의 에바를 장착하여, 냉동과 냉장을 동시에 가동하는 모델

▶ 히팅 기능 : 외부온도가 탑내 온도보다 낮을 때, 제품이 어는 것을 방지하기 위해 핫가스, 라지에타 핫워터 또는 전기코일을 이용, 적정온도를 유지하는 기능. 주로 야채, 과일, 음료 운반시 사용

▶별도 아우어메타 : 컨트롤러 외 컨덴서 사이드 도어에 별도 아우어메타를 장착 엔진가동시간 확인기능
