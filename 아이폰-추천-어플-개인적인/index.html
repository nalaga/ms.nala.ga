아이폰 추천 어플 (개인적인)

  • By admin

  • in 미분류

3줄 요약

  1. 24. iNeedCoffee : 커피전문점을 찾을 때~~
  2. 29. 음악어플 : 엠넷, 벅스, 소리바다 등등..
  3. 무료어플을 정기적으로 소개하는 블로거들의 글을 구독하는 것도 좋습니다.

주변에 아이폰을 사는 사람들이 많아지면서 어플을 추천해달라는 경우도 그에 비례하고 있습니다.

허허.. 이거참.. 처음엔 몇개만 추천하기도하고, 아님 검색창에 ‘아이폰 추천어플’혹은 ‘아이폰 필수어플’로 검색하라고 했는데..
생각해보니 매번 그런 이야기만 하기엔 미안시럽기도하고, 아주 약간 귀찮기도(^^;;)하고..
하여 그냥 제가 쓰는 어플을 다 공개합니다. (다른 좋은 어플도 많으나 내가 쓰고 있는 것만.)
대신 순정폰이란 전제하에, 그리고 모두 무료어플로만 소개할 것입니다. 
(유료어플에 대한 호불호는 자본주의 주류경제학의 ‘효용’에 따라 다르니까 그건 개별적으로 판단을..^^;;)
그리고 상세설명보다 그냥 어플에 대한 간단한 언급만 할 생각입니다. 왜냐면!! 그 정도는 스스로 알아봐야 ^ㅡ^
0. 앱스토어에서 daum, naver, google로 검색해보면 나오는 어플들이 있음. 이건.. 뭐.. 개인의 취향..대로..ㅋㅋ
1. 2Do Lite : 일정관리 어플
2. Evernote : 메모 어플 (온라인계정)
3. SpringNote : 메모 어플 (팀으로 이용시 유용)
4. Air Files : 아이폰을 메모리처럼 쓸 수 있음. (office문서 등을 읽을 수 있음)
5. 세컨드라이브 : 요것도 메모리처럼.. 국내계정이고 1테라바이트 공간을 내어줌. (office문서 등을 읽을 수 있음)
6. Dropbox : 이것도 메모리~~ (office문서 등을 읽을 수 있음)
7. 한컴뷰어 : 한글파일을 읽을 수 있음.
8. twitbird : 트위터 어플 (리스트로 정리해서 읽기에 용이)
9. echofon : 트위터 어플 (타임라인을 한꺼번에 읽기에 용이, echofon 사용자간 푸시 무료)
10. facebook : 페이스북 어플 (ㅋㅋㅋㅋ추억의 인물들을 만날 수 있음. 왕산을 말아먹은 위인이 여럿있음)
11. 카카오톡 : 카카오톡 사용자간 실시간채팅 가능 (그룹채팅이 짱임.. 다른 유료앱이 필요없음)
12. Flickr : 사진을 올리고 공유하는 yahoo서비스
13. tumblr : 블로그 어플
14. foursquare : 위치기반 사회성.. 게임(?) 이건.. 음.. 해봐야 알 수 있음.
15. nateOn : 네이트온 어플
16. Seoul Bus : 버스도착정보와 노선정보를 알 수 있는..
17. Bus Stop : 버스정류장을 찾을 수 있는 ( 이런 비슷한 서비스인데 조금씩 다른 ‘버스노선도’, ‘지하철연계버스’등이 있음)
18. 서울교통정보, 고속도로정보 : 길막히는지 확인할 때 유용
19. 지하철 : 지하철노선도, 시간표 등..
20. iKorail : 열차시간표, 예매가 되던가..??
21. iKobus : 고속버스검색
22. sekai camera : 증강현실 어플. 이것도 해봐야 알 수 있음. 모르겠으면 검색~~ㅋㅋ
23. Layer, ScanSearch, Odiyar : 증강현실 어플.. 뭘 찾을 때 유용.
24. iNeedCoffee : 커피전문점을 찾을 때~~ 
25. Stanza : eBook을 읽을 수 있는 어플..이건 사용법이 좀 복잡하니..여길 참고 http://moonshot.egloos.com/3174935
26. iStardict : 이게.. 이름이 바뀌었는데.. Gurudic으로.. 유료로 바뀌었던 것 같음. 사전 어플임.
                   요것도 사용법이 살짝 복잡하니 잘 정리된 블로그를 소개함. http://eu81273.tistory.com/
27. 라디오 어플 : MBCRadio, SBSRadio, arirangRadio, HotRadio 등등 라디오 들을 수 있는 거 많음.
28. 영상관련 어플 : daumTV팟, 아프리카 등등.. (MoSS, AirVideoFree라는게 있는데 이건 검색창에서 알아보시는게..ㅋㅋ)
29. 음악어플 : 엠넷, 벅스, 소리바다 등등..
30. 뉴스어플 : …한겨레만 빼고 다 있으니 찾아보면 됨. 민중의소리어플이 출시일을 넘겼으나 계속 안나오고 있음. -_-;;
                    (연합뉴스, YTN, 매일경제 등등은 실시간 방송도 제공함)
31. Instapaper, Read It Later : 웹페이지 링크를 저장했다가 나중에 읽을 수 있음.
32. MobileRSS : 구글RSS와 연동되어서 구독 중인 컨텐츠를 읽을 수 있음.  (RSS가 궁금하다면.. 역시 검색..ㅋㅋㅋ)
33. Tatter&Media : 양질의 블로그를 정리해서 뉴스처럼 보여줌. 개인적으로 좋음. 비슷한게 몇개 더 있지만.. 난 별로임.
34. 카메라 어플 : Gorillacam(줌도 되고 뭐..), Action 16(빠르게 16연사가 가능한..), SlowShutter(말그대로..야간촬영에 효과적인지 시험안해봤음), SelfPhoto(셀카질할 때??), All in one camera(찰칵소리안남..), PS mobile(포토샵편집), 그외 다양한 사진효과가 가능한 것들로 ColorsPro, Mill color, PhotoFilters, MoreBeaute, MoreMono, MoreNoel, Polarize, PhotoFunia, stachetastic, Watercolor등이 있음 (다 한번씩 실행시켜봐야 알 수 있음. 그외 많으니 앱스토어 카테고리 활용~)
35. 쇼고객센터와 쇼멤버십카드는.. 알아서 깔겠죠?
36. 은행어플도 있으니.. 자기가 쓰는 은행을 찾아서..ㅋ지금까지 하나, 우리, 신한이 나온 것 같음. 계속 추가될 듯.
37. iBed : 세로보기로 고정된 브라우저(누워서 보기 좋음)
38. 초성변환 : 전화번호부 초성검색을 가능하게 주소록을 수정해줌.
39. 그룹어플 : 나는 contacts G.가 좋음. 업데이트를 자주 해줘서..??
…
의지박약이란 비난을 듣겠지만.. 더 못하겠군요..
아. 너무 졸리고.. ㅠ_ㅠ 그리고 나머지는 진짜 그냥 개인의 취향인 듯..
앱스토어에 카테고리를 잘 둘러보고 사용해보는 것도 좋고,
‘오늘만 무료’나 ‘Hottest Apps’같은 무료어플 소개 어플을 이용해서 알아보는 것도 좋고..
무료어플을 정기적으로 소개하는 블로거들의 글을 구독하는 것도 좋습니다.
예를들어 : 도아의 세상사는이야기  http://www.offree.net
음.. 어차피 스마트폰이라는게.. 제일 유용한 것은 소셜미디어를 이용하는데 있는 것 같고..
나머지 부가적인 기능들은 사용자들의 needs에 따라 선택하는거니깐..
암튼 활용하기 나름인 아이폰이니!! 열심히 잘 사용하시길!!!
