SASS & Frameworks를 활용한 ‘반응형 웹 디자인’ 모던 워크플로우 실무 향상 과정

– 진보된 Sass + Compass + Bourbon + Susy를 활용한 반응형 웹 모던 사이트 제작 워크플로우 –

CSS 중심의 반응형 웹디자인(흔하디 흔한 과정)과는 구축과 운영 측면(생산성, 관리효율)에서 차원이 다른 반응형 웹 모던 사이트 제작 워크플로우를 배우는 과정입니다.

* 수강료 69~86% 정부지원!!
– 교육비: 정규직 182,736원, 계약직 82,170원 (근로자카드 신청자에 한함)
* 근로자 직업능력개발훈련이란?
* 수강료 정부지원시 결제는 사전등록 후 교육당일 근로자카드결제만 지원!!
