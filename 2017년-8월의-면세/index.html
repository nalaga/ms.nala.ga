2017년 08 면세지름

1. 슈에무라 X Blanc:Chroma cushion Puff + 파운데이션 쿠션

 동생의 추천으로 갈아타봤으나 10월 구매한 에이지 트웨니스에 밀려 그대로… 굿베이… 이상하게 동생꺼 썼을 땐 하루종일 수정 화장 안해도 될만큼 좋았는데 사서 쓰니까 걍 무난했다. 커버력이 좋은지도, 지속력이 좋은지도 잘 모르겠으나 촉촉하긴 했던 것 같음. 걍 무난했음. 너무너무 무난.


2. 에스쁘아 #브룸스트리트/브론즈페인팅 펜슬

 참 오랜만에 써보는 깍아 쓰는 아이라이너. 노쌍커플+눈밑번짐이 심한 나에게 번지지도 않고 잘 맞는 라이너. 특히 라이너로 그리고 끝에 달린 스펀지로 문질러 쉐도우처럼 사용할 수 있어 노쌍커플인 내 눈에 잘 맞는 편. 부드럽게 잘 발려서 쉐도우 대용으로 잘 쓰고 있다.

3. 맥 테크니콜 라이너 #포토그라뷰어

 돌려쓰는 아이라이너. 이거두 잘 안번지고 슥슥 잘 그려짐. 하지만 돌려쓰는 것 치고 얇은 편은 아님. 톰포드나 슈에무라랑 비교하면 살짝 두꺼워서 눈꼬리 뺄 때 살짝 애매함. 

4. 톰 포드 뷰티 하이 데피니션 라리하리너 버니쉬드 골드 / 에보니

 정말 부드럽고, 얇고 잘 그려짐. 슈에무라 뺨치게 좋음. 단점이라면 가격… 가격… 박스도 예쁘고, 라이너도 예쁘게 생김. 모든게 완벽하나 가격이 단점임. 하나는 엄마가 쓴다고 해서 엄마 줘서 못 그려봄. 다음에 또 살 예정. 특히 얇아서 눈꼬리 뺄때 참 조음.

5. 메이크업 포에버 #Diamond Brown D-62

 아직 못뜯음. 위에 라이너 3개를 한번에 뜯어버리면서 이 아인 잠시 미개봉인 상태로 보관하는 걸로.

6. 슈에무라 루즈 언리미티드 슈프림 매트 립스틱 #M CR 344

 색이 정말 정말 예쁘나 나에게 안 어울림. 꽃 분홍색. 상큼하고 과즙 메이크업이 잘 어울리는 사람한테 잘 어울릴 거 같음.

7. 톰 포드 뷰티 립스 앤 보이즈 19 제임스 / 35 라파엘

 톰 포트 립스틱 향이.. 더럽… 넘나 톰포드 빠수니 같지만 예쁘고 아름다운 걸 어떡함. ㅠ^ㅠ 제임스는 내 입술에 발색했을 때 분홍색이지만 넘나 자연스러워서 바른거 같지가 않아 살짝 아쉬움. ㅠㅠ 라파엘은 진짜 … 하… 너무 예뿌다……

8. 톰포드 뷰티 화이트 스웨이드 EDP 

 오빠 선물로 산 향수. 향이 고급스러움. 좀 머리아프다 느낄 수 있을 만큼 무겁고 진한 향이긴 하나, 고급진 향이남. 이 향 맡다가 조말론 향수 뿌리면 라이트하게 뿌린 거 같은 느낌까지 남. 대체로 중성적인 향수만 사는 나에게 넘나 마음에 드는 향.


9. 코스알엑스 원스텝 모이스쳐 업 패드

 이거 없이는 세수 했다고 할 수 업서…… 더럽…
