[후쿠오카 여행] 2박3일 1일차 ⑨ 야나가와에서 텐진, 니시진에서 후쿠오카 타워

  • By admin
  • on 1월 16, 2018
  • in 미분류

3줄 요약

  1. 마지막엔 이거 하나 남네요 ㅎㅎ
  2. 이제 시내 지하철을 탈거라
  3. 왼쪽에 있는 정류장으로 향했습니다

야나가와 뱃놀이까지 즐긴 후

다시 역으로 돌아와 텐진으로 향했습니다

 

 

 

 

 

 

셔틀버스타고 되돌아온 야나가와역

칸코킷푸를 보여드리고 들어갔어요

 

 

 

 

야나가와에서 텐진으로는

바로 급행이 옵니다

 

따로 환승하거나 그럴 필요가 없었어요

 

 

 

 

 

 

야나가와에서 텐진까지는

총 37정거장에 850엔

 

저는 급행이라 7정거장만 섰어요

 

 

 

 

 

 

텐진방향은 타는곳 3번

여기가 가장 사람이 많습니다

 

 

 

 

 

 

처음 와본 곳이였는데

나름 잘 즐겼던 야나가와 ㅎㅎ

 

 

 

 

 

 

시간이 남아 조금 기다려야 했어요

 

아직 스크린도어가 없는 일본 지하철은

느낌이 참 좋은 것 같습니다

 

 

 

 

 

 

열차시간표도 찍긴 했는데

열차 간격은 12~15분씩이네요

 

1시간에  4대정도 오니

아주 자주오는 편은 아닌 것 같습니다

 

 

 

 

 

 

텐진(니시테츠 후쿠오카)행

급행 열차를 타고 출발했습니다

 

점점 텐진쪽에 가까워지니

사람이 많아지더라구요 ㅎㅎ

 

 

 

 

 

 

마지막 종점에 내릴 때에는

완전 인산인해였어요

 

그리고 열차 의자가 좀 독특했는데

직접 돌려서 마주보고 갈 수 도 있고,

그걸 다시 돌릴 수도 있었습니다

 

 

 

 

 

 

그렇게 텐진에 도착해서

마지막 칸코 킷푸를 보여드리고 나왔습니다

 

 

 

 

 

 

알차게 사용했던 칸코킷푸

마지막엔 이거 하나 남네요 ㅎㅎ

 

 

 

 

 

 

니시테츠 텐진역에 도착해서

열차를 환승하러 갔습니다

 

 

 

 

 

 

이제 시내 지하철을 탈거라

2층 니시테츠선에서 지하 2층

지하철 탑승구로 갈겁니다

 

 

 

 

 

 

바로바로 연결되어 있어서

참 편리했네요 ㅋㅋ

 

여기에서는 시티패스를 보여주고 입장

 

 

 

 

 

 

저는 후쿠오카 타워로 갈거라서

니시진역으로 갈겁니다

 

텐진역에서 니시진역은 4정거장 260엔

 

 

 

 

 

 

타는곳 2번에서 탑승하면 됩니다

 

 

 

 

 

 

그렇게 텐진역에서 출발해

4정거장 후

 

 

 

 

 

니시진역 도착 !

 

이번에 니시진역도 처음 와보네요

항상 버스로만 다녔어서 ㅋㅋ

 

 

 

 

 

 

그렇게 니시진역에 도착 후

7번 출구로 나왔습니다

 

( 사실 다른 출구로 나갔었는데

여기가 정류장이랑 가장 가까웠던걸로 기억해요 )

 

 

 

 

 

 

그렇게 니시진역에 도착해서

왼쪽에 있는 정류장으로 향했습니다

 

 

 

 

 

 

우동가게랑 주차장을 지나가면

바로 정류장이 하나 보일거에요

 

 

 

 

 

 

바로 여기 !

 

후쿠오카 야후쿠돔?

그런 정류장입니다 ㅋㅋ

 

 

 

 

 

 

후쿠오카 타워로 가는 버스는

10번, 15번, 54-1번, 94번

 

 

 

 

 

 

저는15번 버스가 가장 빨리와서

이걸 타고 갔습니다

 

 

 

 

 

 

버스에도 가장 끝에

후쿠오카 타워라고 적혀있었어요

 

 

 

 

 

 

어차피 시티패스를 보여드릴거지만

뒷문으로 타서 꼭 뽑아야하는 번호표

 

 

 

 

 

 

시간이 어중간해서 그런지

타워쪽으로 가는 버스 안에는

사람이 별로 많지 않았습니다

 

 

 

 

 

 

오늘 하루 알차게 쓴 시티패스

 

있다가 숙소로 돌아갈 때에도

시티패스를 쓸거에요 ㅎㅎ

 

 

 

 

 

 

그렇게 종점인 후쿠오카타워에

도착을 했습니다

 

여기도 거의 1년만에 오는 곳이라

나름 반갑더라구요 ㅎㅎ 익숙하기도 하고

 

 

 

 

 

 

저번에 왔을 때에는 너무 늦게 와서

타워만 보고 갔었는데

 

이번에는  모모치 해변을 보려고

조금 일찍 도착했어요 ㅎㅎ

해변에서 노을보면 예쁠 것 같더라구요

 

그래서 바로 모모치해변으로 향했습니다
