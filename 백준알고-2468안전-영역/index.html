문제

재난방재청에서는 많은 비가 내리는 장마철에 대비해서 다음과 같은 일을 계획하고 있다. 먼저 어떤 지역의 높이 정보를 파악한다. 그 다음에 그 지역에 많은 비가 내렸을 때 물에 잠기지 않는 안전한 영역이 최대로 몇 개가 만들어 지는 지를 조사하려고 한다. 이 때, 문제를 간단하게 하기 위하여, 장마철에 내리는 비의 양에 따라 일정한 높이 이하의 모든 지점은 물에 잠긴다고 가정한다.

어떤 지역의 높이 정보는 행과 열의 크기가 각각 N인 2차원 배열 형태로 주어지며 배열의 각 원소는 해당 지점의 높이를 표시하는 자연수이다. 예를 들어, 다음은 N=5인 지역의 높이 정보이다.

이제 위와 같은 지역에 많은 비가 내려서 높이가 4 이하인 모든 지점이 물에 잠겼다고 하자. 이 경우에 물에 잠기는 지점을 회색으로 표시하면 다음과 같다. 

물에 잠기지 않는 안전한 영역이라 함은 물에 잠기지 않는 지점들이 위, 아래, 오른쪽 혹은 왼쪽으로 인접해 있으며 그 크기가 최대인 영역을 말한다. 위의 경우에서 물에 잠기지 않는 안전한 영역은 5개가 된다(꼭지점으로만 붙어 있는 두 지점은 인접하지 않는다고 취급한다). 

또한 위와 같은 지역에서 높이가 6이하인 지점을 모두 잠기게 만드는 많은 비가 내리면 물에 잠기지 않는 안전한 영역은 아래 그림에서와 같이 네 개가 됨을 확인할 수 있다. 

이와 같이 장마철에 내리는 비의 양에 따라서 물에 잠기지 않는 안전한 영역의 개수는 다르게 된다. 위의 예와 같은 지역에서 내리는 비의 양에 따른 모든 경우를 다 조사해 보면 물에 잠기지 않는 안전한 영역의 개수 중에서 최대인 경우는 5임을 알 수 있다. 

어떤 지역의 높이 정보가 주어졌을 때, 장마철에 물에 잠기지 않는 안전한 영역의 최대 개수를 계산하는 프로그램을 작성하시오. 

입력

첫째 줄에는 어떤 지역을 나타내는 2차원 배열의 행과 열의 개수를 나타내는 수 N이 입력된다. N은 2 이상 100 이하의 정수이다. 둘째 줄부터 N 개의 각 줄에는 2차원 배열의 첫 번째 행부터 N번째 행까지 순서대로 한 행씩 높이 정보가 입력된다. 각 줄에는 각 행의 첫 번째 열부터 N번째 열까지 N 개의 높이 정보를 나타내는 자연수가 빈 칸을 사이에 두고 입력된다. 높이는 1이상 100 이하의 정수이다. 

출력

첫째 줄에 장마철에 물에 잠기지 않는 안전한 영역의 최대 개수를 출력한다. 

예제 입력 

5
6 8 2 6 2
3 2 3 4 6
6 7 3 3 2
7 2 5 3 6
8 9 5 2 7

예제 출력 

5

문제를 파악해보면.. 안전영역이란 안전한 구역의 집합들 입니다. 먼저 N을 입력받고 N*N 배열이 생깁니다. 그 후 건물의 층을 입력받고, 0부터 가장 큰 건물의 수 높이까지 만큼 비가 내렸을때 안전영역이 가장 많은부분을 구하는겁니다.

제 경우에는 침수된 부분을 0으로 두었고, 침수가 되지 않은 부분을 1로 두었습니다.

이런식으로 배열이 만들어져서 1로 묶인부분인 안전영역이 4가 될것입니다.

이렇게 높이가 0, 1, 2, , , , , 9까지 비가 왔을때 가장 안전한 영역이 많은것의 수를 구하는것이 이 문제의 출력문이 되겠습니다.

#include<iostream>
using namespace std;
#define MAX_VALUE 101
int map[MAX_VALUE][MAX_VALUE];        //실제 맵
int compMap[MAX_VALUE][MAX_VALUE];    //침수지역은 0 비침수 1로 설정해놓을배열
int N;
int dx[4] = {1, –1, 0, 0};
int dy[4] = {0, 0, –1, 1};
 
void dfs(int y, int x) {
    
    if(compMap[y][x] == 1)
        compMap[y][x] = 0;
    else
        return;
        
    if(x < 0 || y < 0 || x >= N || y >= N)
        return;
 
    for(int i=0; i<4; i++) {
        int nx = x + dx[i];
        int ny = y + dy[i];        
 
        dfs(ny, nx);
    }
}
 
int makeMap() {        //맵을 만들고 최대높이의 건물 리턴
    cin >> N;
    int mxf = 0;
 
    for(int i=0; i<N; i++) {
        for(int j=0; j<N; j++) {
            cin >> map[i][j];
            mxf < map[i][j] ? mxf = map[i][j] : mxf;
        }
    }
    return mxf;
}
 
void makecompMap(int rainFloor) {    //들어온 rainFloor인자보다 작으면 0, 크면 1로 compMap을 설정
    for(int j=0; j<N; j++) {
        for(int k=0; k<N; k++) {
            if(map[j][k] <= rainFloor)
                compMap[j][k] = 0;
            else
                compMap[j][k] = 1;
        }
    }
}
 
int countSafety() {        //안전한 영역 구하기 , 배열 완전탐색으로 침수 안된 부분을 
                        //dfs()를통해 1로바꾸고 dfs호출수만큼 Area증가시켜서 리턴하는 함수
    int safetyArea = 0;
 
    for(int i=0; i<N; i++) {
        for(int j=0; j<N; j++) {
            if(compMap[i][j] == 1) {
                dfs(i,j);
                safetyArea++;
            }
        }
    }
    return safetyArea;
}
 
 
int main() {
    int maxFloor = makeMap(); //맵을 부여하고, 최대높이를 구함
    int maxValue = 0;
        
    for(int i=0; i<=maxFloor; i++) {        //TestCase를 기준으로 0부터 9까지 반복됨.
        makecompMap(i);                    //맵을만들고
        int temp = countSafety();        //안전영역을 temp에다 넣은후
        maxValue < temp ? maxValue = temp : maxValue;    //안전영역 중 최대값을 maxValue에 넣음
    }
    
    cout << maxValue;    //안전영역 중 최대값인 maxValue를 출력
    return 0;
}

다음으로는 makecompMap(int rainFloor) 함수 내용입니다. …함수 이름이…하하 … 경험이 많지 않아서… 함수이름을 저런식으로 ㅎㅎ.. 비교할 맵(?) 이런식으로 이름 지어봤어요…ㅎㅎ

어쨌든, 매개변수 rainFloor로 들어오는 값은 비가 오는 높이값이 들어오게 됩니다.

이중 for문을 통해서 배열의 완전탐색으로 맵을 탐색합니다.  탐색중에 비가 온 높이보다 건물크기가 작거나 같으면 0 값을 compMap이라는 배열에 같은위치에 넣어주고, 비가 온 높이보다 크면 1 값을 넣어줍니다.

즉, 이 함수는 (0 ~ 비가 올 최대높이) 번 사용됩니다.

마지막으로 countSafety() 함수입니다. 안전한 부분을 count 해주는 함수죠..

바로전 makecompMap 함수에서 설정된 배열을 기준으로 이 부분에서 안전영역의 개수를 구해주고, 리턴하는 함수입니다.

이거또한 compMap 배열을 완전탐색 하면서, 침수가 안된 지역이 있으면 그부분을 1로 만들어줍니다.

그리고 여기서 이 문제에 핵심인 dfs(i,j)를 넣어줌으로써 침수가 안된지역 주변의 침수 안된지역까지 모두 1로 만들어 주는겁니다.

그렇지만 safetyArea 변수는 dfs가 진행되는동안 1번만 증가시켜 줌으로써 완전탐색 후 정확히 안전한 영역만 safetyArea값으로 들어가게 되겠죠.
