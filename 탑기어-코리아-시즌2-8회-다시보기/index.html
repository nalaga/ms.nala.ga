탑기코 MC들이 차량의 도어에 대해 잘 모르는것 같아서 써봅니다. 

OEM (Origanal Epuipment Manufacturer) 

:이건 일반차에 적용된 그냥 평범한 문이라고 생각하시면 됩니다. 가장 대중화되어 있고 대부분의 차가 이 문을 사용하고 있기도하죠…  레귤러 도어라고도 하고 스윙도어라고도 합니다. 

 

다시보기 영상은 여기서 ▶▷ http://blog.naver.com/reidea/30139651319

슬라이딩 도어( Sliding) 

: 슬라이딩 도어는 말 그대로 슬라이딩하여 여는 문입니다. 보통 밴이나 승합차에 이런문이 많이 달려있죠… 

벤츠 스프린터 

걸윙도어 (Gullwing) 

: 걸윙도어는 문이 갈매기의 날개와 비슷하다고 하여 붙혀진 이름입니다. 걸윙도어는 문이 수직으로 열리는 방식입니다. 

걸윙도어는 벤츠 SLS나 파가니 후에이라 같은 차에서 볼수있는 독특한 문입니다. 

 

벤츠 SLS 

파가니 후에이라 

시저도어 (Scissor) 

:시저도어는 가위처럼 열린다고하여 붙혀진 이름입니다. 일반적으로 람보르기니에서 많이 볼수있는 도어입니다. 그래서 람보도어 혹은 만세도어라고 불립니다. 

람보르기니의 쿤타치 디아블로 등등에 장착되어있습니다. 시저도어도 수직으로 열리기는 하지만 걸윙도어와는 다르게 열리는 모습을 볼수있습니다 

  

 람보르기니 아벤타도르 

  람보르기니 디아블로 

버터플라이 도어 ( Butterfly ) 

:버터플라이 도어는 나비의 날개와 비슷하다고 하여 붙혀진이름입니다. 버터플라이 도어는 벤츠 SLR이나 엔초페라리 멕라렌 F1 등등에 장착되어 나오는 도어입니다. 시저도어와는 비슷하지만 문의 측면이 앞쪽을 바라본다는게 차이점입니다. 

엔초페라리 

멕라렌 F1 

수어사이드 도어 ( Suiside ) 

: 수어사이드 도어는 마차에서 유래된 도어형태입니다.  롤스로이스의 팬텀이나 고스트에서 볼수있는 도어인데요. 차량의 문이 양방향으로 열리는도어입니다. 이 수어사이드 도어는 차 중앙에 B필러가 없기때문에 타고 내리기가 쉽다는 장점이 있습니다. 수어사이드도어는 B필러가 있는 차도 있습니다.  차에 따라서 코치도어라고 부르기도 합니다.

 

롤스로이스 고스트 

다이 히드럴 힝크로 셀릭스 도어 ( Dihedral Synchro Helix ) 

: 이는 코닉세그에서 자체개발한 도어로 시저도어에서 좀더 발전된 형태의 도어입니다. 시저도어와 비슷하게 수직으로 열리지만 문이 열릴때 옆으로 문이 톡 튀어나와 그자리에서 90도 회전하는 형태의 문입니다. 이문의 장점은 차 문이 위로 높이 올라가지 않는게 장점입니다. 

코닉세그 CCXR 

아제라 R 

캐노피 도어 ( Canopy ) 

: 캐노피 도어는 전투기 조종석 처럼 열린다 하여 붙혀진 이름입니다. 현재 이런 도어를 사용하는 차는 거의 없고 대부분 컨셉카에 이용되는 도어입니다. 

120603 탑기어 코리아 시즌2 8화 다시보기 김진표씨나 연정훈씨가 아벤타도르나 엔초페라리 무르시엘라고 도어를 보고 “돡치고 걸윙도어임~” 하는거보고 하도 어이없어서 글씁니다. 전직 레이서에다가 자칭 전문인이라는 사람들이 이런거 하나 모르면서 탑기어 MC를… 정말 어이없다는… 

탑기어의 MC라면 일반인이 알고있는 정보보다 더 자세하고 알려지지않은 정보를 알려주는 사람들인데 일반인보다 못한 지식으로 MC하고있는… 

차량 팜플렛 줄줄이 외워서 방송하고 있다는걸 확인시켜주는 3MC들… 제발 대본쓸때 작가랑 상의해서 대본 쓰시길…
