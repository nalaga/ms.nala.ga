# 기본형 변수(Primitive Type)
> 기본적으로 크기가 작음, 사이즈가 고정 되어있음
– 논리형 : boolean(1)
– 정수형 : byte(1)/short(2)/char(2)/int(4)/long(8)
– 실수형 : float(4)/double(8)

 

 

# 참조형 변수(Reference Type)
> 사이즈가 고정되어있지않고 가변적임(ex,String)
> 사이즈가 커질수있고 큼(무한개급)
– 기본형을 제외한 나머지 자료 형
– 객체의 주소를 저장한다.
– String 등 참조형 변수는 모두 4byte

  b = tmp; // swap 기법
