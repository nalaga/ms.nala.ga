3줄 요약

  1. 제가 사용하면서 느낀 삼성건조기 단점도 적절히 녹여서
  2. 세탁망에 넣은 그대로 돌렸더니 완전 구깃구깃ㅋㅋㅋㅋ
  3. 이렇게까지 구겨지지 않았었는데

2018 삼성건조기 인버터 9kg
DV90M53A0QX  플래티넘 이녹스

직접 2주 동안 열심히 사용해 보고 후기 올려봐요
테스트 해본다고 정말 하루 1-2번 이상 사용한 것 같아요

 

 

 

 

 

 

소음도 차단하고 배수호스 꽂아서 자동물빠짐기능 사용하려구요ㅎㅎ
물통비우는게 귀찮아서 베란다 설치 ‘-‘

 

구입하기 전에 다른 블로거분들 글 정말 여러번 보고 괜찬다 싶어
바로 구입결정 했는데, 2주 사용해보니 만족도 높아요!!
제가 너무 좋다고 하니 신랑왈
“내가 진작에 건조기 사자고 했지?”
ㅋㅋㅋㅋㅋㅋ “응…ㅋㅋ”

그치만 단점이 없는건 아니더라구요.
제가 사용하면서 느낀 삼성건조기 단점도 적절히 녹여서
리뷰 시작해 볼게요.

 

 

 

 

삼성건조기 : 소음
삼성건조기 9kg
삼성건조기 인버터 동영상으로 직접 소리 들어보세요
이거 실내에 설치했으면 좀 시끄러웠을 것 같아요.
전 베란다라서 문 닫으면 방해될 정도는 아니라서 괜찮은데,
혹시 옷방이나 거실쯤 설치하시면 소음때문에
불편할것 같아요 왠만한 세탁기 돌아가는 소리랑 비슷해요
(그리고 옷에 혹시 딱딱한부분이 있으면 돌때마다 달그락 달그락 소리도 남 은근 거슬림ㅋㅋ)

 

 

 

삼성건조기 : 수건건조

 

 

 

 

 

정말 이 먼지 우리집 수건에서 나온거 맞나요?
나름 깔끔한 성격때문에 빨래 정말 열심히 했었는데

수건에서 먼지가 한가득 나와서 오마이갓..
너무 놀랐던 첫번째 건조
실화냐?ㅋㅋㅋㅋㅋㅋㅋㅋ

일단 첫 사용부터 먼지 한가득 나오는거 보고 일단
삼성건조기 신뢰
ㅋㅋㅋㅋㅋㅋ

수건은 주변에서 말하듯 보송보송
피부에 닿는 느낌이 보드러워졌어요

이후로도 계속 돌려보니 수건은 돌릴때마다 머리카락이며
먼지며…난리도아니에요 ㅠㅠ

수건은 100점ㅋㅋㅋ

 

 

 

삼성건조기 : 이불건조
이불 건조는 쏘쏘…

 

 

 

 

 

 

 

 

이불 덮는거 까는거 두개 빨아서 구겨넣고
건조기 1번 돌렸는데, 엥? 반밖에 안마름.

너무 양이 많은가 싶어서 1번 더 돌림
엥? 극세사는 끝부분이 안마름 뭐지?ㅡㅡ

찾아봤더니 내가 이불을 너무 많이 넣었던 것…
이불건조는 2kg 까지만 잘 된다고 하더라구요
너무많이 넣었더니 잘 안되요 ㅋㅋㅋㅋㅋㅋ

건조기에서 꺼내 반나절 건조대에 말려줬어요

이불은 얇은이불 1개정도 건조시키거나
아니면 이불털기 기능으로 먼지터는 정도만 사용하는게 좋을 듯 싶어요

 

 

 

 

삼성건조기 : 이불털기
마른 이불을 이불털기 모드로 돌려봤어요 50분 소요
역시 내가 욕심쟁이인건지 겨울이불 구겨넣음
두꺼운 이불은 먼지가 생각보다 안털림
한번 빨아서 그런가? 먼지가 생각보다 안나오더라구요

 

 

 

 

 

 

 

먼지도 어느정도 나오긴 했어요.
근데 수건처럼 비포 애프터가 확실한지는 잘 모르겠어요

얇은 이불은 확실히 다르려나?
이건 다음에 해보는걸로 ㅎ.ㅎ

 

 

 

 

삼성건조기 : 아기옷 & 손수건

곧 출산이라 아기옷, 아기손수건(거즈,면)
열심히 빨아서 건조대에
햇빛 쨍쨍 소독시켰었는데

건조기 들이고 다시 건조기에 이불털기 기능으로
손수건이랑 아기옷 50분 돌려봤어요

 

 

 

 

 

 

아기손수건 특히 거즈손수건 에서 먼지가 이렇게 많이 나오는건지 ㅜㅜ
면역력 약한 아기에게 이렇게 먼지많은 손수건을 사용하게 될뻔..!!
수건, 아기옷, 아기손수건 등은
건조기 정말 혁명인것 같아요

 

 

 

 

 

 

 

너무 당황했어요 앞으로 아기옷은
무조건 건조기로 먼지털기 한번씩 해주는걸로!
살균도 되고 먼지도 털고 보송보송
손에 닿는 느낌도 차원이 다르네요

 

 

 

 

 

 

 

 

 

 

 

 

삼성건조기 : 셔츠

 

 

 

 

 

 

그리고 세탁망에 넣고 빨았던 신랑 셔츠
세탁망에 넣은 그대로 돌렸더니 완전 구깃구깃ㅋㅋㅋㅋ
셔츠는 건조하더라고 세탁망에 넣음 안되겠어요

세탁망에 안넣고 돌렸던건
이렇게까지 구겨지지 않았었는데

한 이주정도 직접 사용해보니
수건, 티셔츠, 레깅스, 양말
일상적인 세탁물들은 건조하기 너무 좋네요

단지 이불 건조가 2kg 이 한계라는점이 조금 아쉽고

이불건조 빼곤 전 너무 좋은 것 같아요
아직 한달내내 사용해서 전기세 확인 안해봤지만

전기세좀 나온다쳐도 거실이나 방에
빨래건조 안해도되서
먼지가 확실히 적어지긴 했어요
