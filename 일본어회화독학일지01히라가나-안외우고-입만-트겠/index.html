이렇게 넘기니 DAY 01과 02를 한꺼번에 공부해도 편안합니다!

오늘 공부한 것 ( 어휘에 맞게 띄어쓰기. 오로지 내가 알기 편하게! )

DAY01

아나따 와 각세- 데쓰까?     =     당신 은 학생 입니까?

하이! 와따시 와 각세- 데쓰      =      네! 저 는 학생 입니다.

이-에, 와따시 와 각세- 쟈 아리마셍     =     아뇨, 저 는 학생 이 아닙니다.

~데스     ->     ~입니다.

~데스까?    ->     ~입니까?

아나따 와     ->     당신 은

와따시 와     ->     저 는

각-세     ->     학생

하이     ->     네

이-에     ->     아뇨

~쟈 아리마셍     ->     ~이 아닙니다.

DAY02

코레 와 난 데스까?     =     이것 은 무엇 입니까?

소레 와 한코쿠 노 혼- 데쓰     =     그것 은 한국어 책 입니다.

아레 와 아나따 노 데쓰까?     =     저것 도 당신의 것 입니까?

소레 와 와따시 노 토모다찌 노 혼- 데쓰     =     그것 은 저의 친구의 책 입니다.

코레 와     ->     이것 은

난     ->     무엇

소레 와     ->     그것 은

한코쿠 노 혼     ->     한국어 의 책

~노     ->     ~의(혹은 ~의 것)

아레 와     ->     그것 은

가-루후렌드     ->     여자친구(걸프렌드)

토모다찌     ->     친구

카반     ->     가방

노토     ->     노트

확실히 일본어가 쉽다. 한국어랑 어순이 비슷
