        태그

안녕하십니까.

15.10.02 좋은 아침입니다. 오늘은 바람이 어제보다 많이 부니 다치지 않도록 조심하십시오

 

오늘은 HTML을공부하기 이전에 동기부여를 하고 무슨페이지를 제작할건지 생각하고 제작에 임하도록 하겠습니다.

 

 

왼쪽에는 메뉴 오른쪽에는 콘텐츠 영역들이 들어가도록 디자인을 해보도록 하겠습니다.

(2단 레이아웃 페이지)

 

오늘은 첫 강좌니깐 간단하게 HTML에 존재하고 자주사용하는 코드들만 간단하게 보고 넘어가도록 하겠습니다.

 

 

 

<html> 시작해서 </html> 로 끝나는것이 바로 HTML이다..

작성시에는 주의사항으로는 모두  소문자로 선언하고 사용하는것을 준수하고 있다.

 

위의 코드를 보게되면

<html>
   <head>

   <!– 타이틀 부분과 각종 연결을 할수있는 부분 또는 style을 선언할수 있는 부분 –>

   </head>

 

   <body>

   <!– 실제 html 코드를 삽입하여 페이지 에 보이도록 할수있는 공간 –>

   </body>

</html>

 

HTML에서 사용되는 코드들은 많지만 우리들이 사용하는것은 한정적일수 있습니다.

 

간단하게 많은사람이 사용할것이라고 예상되는 아니 사용하는 코드를 알아보도록하겠습니다.

[코드 == 태그] 라고생각하시면 되겠습니다. 또한 아래의 태그는 <body> 속 삽입태그입니다.

 

<p> 태그

  <p>태그는 간단하게 생각하면 글을 보여주는 태그라고 생각히시면됩니다.

  이태그를 이용하여 글을 작성할시는 자동적으로 한줄이 개행되고 난후 한번더 개행

  예) <p> 한줄이 개행된후 한번더 개행 태그 </p>

 

<br> 태그

   <br>태그는 한줄을 개행해주는 태그입니다 <p>태그와차이는 한줄만 개행한다고 생각 하면 된다.

   예) 한줄만 개행되는 태그 <br /> or <br>

   <br> 태그는 다른 태그와달리 앞에 도입부가 없는것을 생각해두자.

 

<h> 태그

   <h>태그는 헤딩태그라고 불리며 글씨의 크기가 이미정해져있는 태그라고 생각하면 간단하다.

   <h1> ~ <h6>까지 사용이가능하다.

   예) <h1> 헤딩원 </h1>

 

<img> 태그

   <img>태그는 앞의 img만 봐도 알다시피 이미지를 불러오는 태그라고 생각하면 아주간단하다.

   예) <img src=”주소” width=”xx” height=”yy” /> 이런식으로 사용이 이루어진다. 

 

<table> 태그

   <table>태그는 테이블을 나누어주는 태그이며 CSS를 사용하지 않고 페이지를 작성시에는 쓴다.

   하지만 거의 .. 안쓴다. 내개인적인 것일수도있지만 HTML을 하다보면 CSS로 디자인을 덮고

   JavaScript로 동적인 페이지를 구성하는것을 목표로 두자..

   예) <table>

<tr> (행의 역할을 담당한다.)

<td> (열의 역할을 담당한다.) </td>

<tr>

  </table>

 

<div> 태그

   <div>태그가 css를 사용하여 페이지제작시에 페이지를 나누는 작업에 자주사용되는 태그이다.

   우리는 이태그를 이용해서 페이지를 제작하도록 할것이기 때문에 이태그는 꼭 기억해야한다.

   예)<div> div 태그 </div>

 

<a> 태그

   <a>태그는 링크를 거는 태그라 생각하면된다. 한글이나 엑셀에서 보면 하이퍼링크가 있다.

   동일한 기능을 하는 태그이다.

   예) <a href=”원하는 주소” target=”창을 어떤식으로 뛰울지여부”>이곳을 클릭시 이동한다.</a>

   자세한 기술사용은 HTML강좌시 필요할때 즉시즉시 알려드리도록 하겠습니다.

 

<ul>, <ol> 태그

   이두개의 태그는 리스트를 작성하는 태그라 생각하면 아주간단하다.

   <ol> 태그의 경우는 순서가 없는 태그라 생각하면 된다.

   <ul> 태그는 순서가 존재하는 태그라고 생각하며 된다.

   주로 페이지를 작성하다보면 <ol> 태그 보다는 <ul> 태그를 많이 사용하게된다.

   예) <ul>

<li>리스트1</li>

        </ul>

   위와 같이 사용하며 ol태그도 비슷하게 작성하여 사용한다.

 

그외에도 많은 태그들이 존재하지만 간단하게 내가 자주사용하는 HTML태그들은 이정도 된다.

 

 그렇다면 메뉴와 컨텐츠 두가지 영역으로 나누는 페이지를 아래의 태그를 이용해서 작성하자.

<table> 태그와 <div> 태그를 이용해서 작성해 보도록 하자.

 

 

           

위의 것이 <table> 태그로작성한 코드                 위의 것이 <div> 태그로 작성한 코드

 

하지만 <div> 태그작성시에는 우리에게 보이는것은 그냥 떨어져 있는 태그로 보일뿐이다.

하지만 위에서 태그 설명을 보면 알다시피 css를 이용하여 스타일을 디자인하면

더욱 깔끔하게 나누어서 처리가 가능하다.

 

CSS 강좌는 추후 업로드해서 알려드리도록 하겠습니다.
