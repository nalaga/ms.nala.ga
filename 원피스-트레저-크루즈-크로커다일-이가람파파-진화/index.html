3줄 요약

  1. 해마신 + 검은 해적 펭귄 + 무지개 해적 펭귄 + 검은 장갑 바닷가재 + 무지개 줄무늬 파수룡
  2. 해마왕자 + 노란 도둑 펭귄 + 노란 투구소라게
  3. 궁금하신 것이 있으시면 언제든지 답변 주세요~

안녕하세요?

 

오늘은 크로커다일과 이가람파파의 진화재료와 평가를 알려드릴게요.

 

원피스 트레저 크루즈를 하다보면 자꾸 성격을 버리게 되는데요.

 

꼭 보스전을 할때마다 밧데리가 없다고 알림이 오는 등 참 짜증이 납니다.ㅋㅋ

 

 


 

 

 

먼저 크로커다일의 진화재료와 순서를 알려드리고, 평가를 해보도록 할게요.

 

크로커다일의 진화는 특이한 것이 3성에서 4성과, 5성 중 하나로 진화가 되는데요.

 

3성에서 4성, 4성에서 5성이 아니라 3성에서 4성이나 5성 둘다 진화가 가능하다는 것입니다.

 

5성 크로커다일은 지덱에 선원으로 활용도가 나쁘지 않은 편인데요.

 

5성은 왕하 칠무해 크로커다일로서 스탯 자체도 평균보다는 조금 나은 편이랍니다.

 

 

 

 

바로크 워크스 사장(3성) => 미스터 제로 크로커다일(4성) 진화재료

 

해마왕 + 검은 해적 펭귄 + 검은 투구소라게 + 무지개 투구소라게 + 검은 줄무늬 파수룡

 

 

 

 

바로크 워크스 사장(3성) => 왕하 칠무해 크로커다일(5성) 진화재료

 

해마신 + 검은 해적 펭귄 + 무지개 해적 펭귄 + 검은 장갑 바닷가재 + 무지개 줄무늬 파수룡

 

 

 

 

크로커다일은 개인적으로는 먼가 아쉬운 캐릭터인데요.

 

3성과 4성의 크로커다일은 독침이라는 필살기로 대미지를 주는 것외에는

 

5성에 비해 좋을 것이 없답니다.

 

5성 필살기는 자신의 속성을 지속성으로 교체하고 대미지까지 주기에 역덱을 맞아 사용하기 용의하답니다.

 

 

 

 

다음은 알라바스타 왕국의 근위대장 이가람인데요.

 

2성과 3성 캐릭터라 활용도는 없는 캐릭터입니다.

 

그래도 우리 비비 공주님을 위하는 마음이 기특하여 넣어봤습니다.

 

 

 

 

MR.8 => 이가람파파 진화재료

 

해마왕자 + 노란 도둑 펭귄 + 노란 투구소라게

 

 

 

지금까지 원피스 트레저 크루즈의 크로커다일과 이가람파파의 진화재료를 알려드렸고,

 

각 캐릭터들에 대한 간단한 평가도 적어보았는데요.

 

궁금하신 것이 있으시면 언제든지 답변 주세요~
