미생물 생화학 기출 예상 정리 (간단한 4문제만)

  • By admin
  • on 1월 13, 2018
  • in 미분류

3줄 요약

  1. 3. 다당류의 기능
  2. ▷ 당질피질 : 다른 세포와 결합, 수용체의 역할
  3. ⓓ 검사 (Inspection)

1. 인간의 미생물 이용
  ⓐ 생명공학
   ▷ 효모의 이용, 페니실린, 효소 등등
  ⓑ 유전공학
   ▷ 재조합 DNA(의학, 공학 농학적 사용)
  ⓒ 생물학적 정화
   ▷ 미생물의 환경 도입, 미생물의 분해 능력으로 오염된 것 정화
 
2. 과학적 방법
  ⓐ 연역적 방법
   ▷ 가설(Hypothesis) → 예측 → 검증 → 결과 → 이론 → 법칙
 
3. 다당류의 기능
  ⓐ 구조물
   ▷ 긴 섬유상 중합체 : 셀룰로오스(Cellulose), 분해가 잘 되지 않음
   ▷ Dextran : 포도당 중합체, 치아 점착성 층 형성
   ▷ 한천(Ager) : 고체배지의 재료, 갈락토오스와 당의 중합체
   ▷ 키틴 : 일부 진균류의 외골격, 글루코사민의 중합체
   ▷ 펩티도글리칸 : 펩티드조직과 다당류의 화합물
   ▷ 지질 다당체 : 그람음성균의 세포벽에 위치, 발열과 쇼크의 원인
  ⓑ 방어
   ▷ 당질피질 : 다른 세포와 결합, 수용체의 역할
   ▷ 탄수화물 : Antibody의 주요 성분
   ▷ 일부 바이러스 : 표면에 당단백질, 숙주세포와 결합, 침투용이
   ▷ Poly saccharoid : 혈액형의 차이 생성
  ⓒ 영양 에너지의 저장
   ▷ 동물세포, 원생동물 : 클리코겐으로 저장
   ▷ 식물세포, 미세조류 : 녹말
   ▷ 가수분해에 의해 중합체 결합 분해
 
4. 미생물의 배양법
  ⓐ 접종 (Inoculation)
   ▷ 시료의 성격에 따른 접종법 선택 (고체 도말, 액체 접종)
   ▷ 성장을 촉진하는 영양분 + 무균상태의 배지
  ⓑ 배양 (Incubation)
   ▷ 성장에 적당한 온도와 환경제공
   ▷ 수시간, 수일, 수주에 걸쳐 미생물 증식
  ⓒ 분리 (Isolation)
   ▷ 존재하는 미생물 세포를 서로분리 하는 것
   ▷ 개별적인 콜로니 형성을 위한 확산평판법, 혼합평판법, 도말평판법 등을 사용
  ⓓ 검사 (Inspection)
   ▷ 콜로니의 특성을 육안으로 보이는 것부터 현미경으로 보이는 것까지 분석하는 것
  ⓔ 동정 (Identification)
   ▷ 미생물을 형태학적, 생리학적, 유전학적, 혈청학적인 특징을 연관짓는것
