20살 처음으로 다녀온 제주도! 짐을 꾸리면서 부터 시작된 여행에 대한 설레발.

여행 기간 : 2017.12.25(월) ~ 2017.12.28(목)

비행기 : 부산 -> 제주도 (에어부산)

           제주도 -> 부산(제주항공)

   = > 왕복 약 75,000원

숙소 : 2박 서귀포 미도호스텔 , 1박당 8인실 도미토리 사용 2만원

        1박 애월 게스트하우스, 2인실 3만원

경비 : 공동경비 15만원, 카메리아힐 네이버 예약, 애월 게스트하우스에서 파티 비용 15,000원, + 45,000원 더 씀.

짐 옮김 서비스 : “가방을 부탁해” 이용

DAY – 1 (월)

공항 도착 13:00

택시타고 만복이네 김밥 1인분 9개 6,500원(먹을 공간이 없어 저는 길거리에서 먹었어요.)

만복이네 김밥 -> 성산일출봉 <201번 버스> 정거장 잘못 내려 광치기 해변 구경!

      광치기 해변 주위에 유채꽃도 있어요!

광치기 해변 -> 성산일출봉 : 정거장은 별로 없는데 정거장 사이 거리가 너무 멀어 걸어서 약 40분.

성산일출봉 도착  너무 추워 파리바게뜨에서 성산일출봉 구경만 함ㅠㅠㅠ

성산일출봉 -> 숙소 , 숙소 도착해서 게스트하우스 사람들이랑 밥먹으러 나감 메뉴는 흑돼지!!(가격은 N/1)

DAY – 2 (화)

09:00시 조식 먹고 출발

숙소 -> 카멜리아힐 : 제가 갔을 땐 동백꽃이 만개하진 않아서 아쉬웠고 포토존은 많은데 그만큼 사람도 많아요! 

                                  가족도 많고 커플도 많아요… 구경 할 공간도 많고 넓어요

카멜리아힐 -> 오설록, 이니스프리 제주 : 택시타고 이동(약 9,000원)

     오설록: 사람이 너무 많았고 차 판매하는 공간이랑 카페있는 공간이랑 같이 있어 사람이 더 많아 보이는 것 같아요..

     오설록에서 굳이 음료를 안마셔도 된다고 하면 2분만 걸어가면 이니스프리 제주에도 다양한 메뉴 있어요!

     이니스프리 제주 체험 : 비누 만들기 체험, 체험 KIT(15,000원)를 사서 만들고 1주일뒤에 사용하 실 수 있어요!

오설록 -> 대기정 : 갈치조림, 2인분 68,000원 (맛있었고 직원분이 너무 친절했어요!)

대기정 -> 서귀포매일올레시장 : 숙소에서 5분 거리에요. 초콜릿 뿐만 아니라 다양하게 먹을거리가 풍부했어요!

                                          우도 땅콩 카라멜 샀는데 몇 봉지 더 사올걸 이라고 후회되더라고요!

 

DAY – 3 (수)

숙소 -> 성산일출봉 : 첫째 날 올라가지 못했던 성산일출봉을 다시 갔어요! 민증보여주시면 24세 까지 청소년이라 입장료 1,000원!

                            왕복 1시간이면 충분하다고 적혀있음. 그치만 저는 올라가다가 한 5번 쉬었어요. 올라가면 경치가 너무 예뻐요.

                            내려갈 때 다리에 힘 풀려서 떨리고 뒤에서 바람 불고 아무 생각 없이 내려와요.

성산일출봉 – > 점심 : 일출봉 앞에 있는 식당에서 전복뚝배기 먹었어요 (1인당 15,000원) – 진짜 맛있었네요. 

성산일출봉 -> 몽상드 애월 : 바다 앞에 있는 카페라 사람이 진~짜 많았어요. 사진 찍는 사람도 많고 카페 안 자리가 너무 붙어 있어요.

                                     몽상드 애월에서 녹차케익 (이름이 기억 안나요 ㅠ) 먹었는데 오설록보다 맛있었어요.

몽상드 애월 -> 캔들 속 제주 : 캔들 파는 가게에요. 기념품으로 사가도 좋을 것 같아요!

캔들 속 제주 -> 숙소 : 걸어서 10분

DAY – 4 (목)

10:45분 비행기

숙소 -> 제주공항 : 택시타고 약 12,000원

다시 일상으로 복귀

그래도 제주도 경치가 너무 이뻐 추웠지만 한번 더 가고싶네요!!
