1줄 요약

  1. 안녕하세요 오늘은 롤 시즌8 이블린 템트리및 룬에 대해서 알려 드릴려고 …중략… 다6.스펠의 경우에는 강타 + 점멸을 고정으로 사용을 해주시면 되겠습니다

안녕하세요 오늘은 롤 시즌8 이블린 템트리및 룬에 대해서 알려 드릴려고 합니다

이번시즌에 들어와서 이블린 정글의 경우 암살자 역할을 더욱더 강력하게 해줄수

가 있다고 생각이 되겠습니다 일단은 6레벨이후에 은신을 통해서 강력한 갱킹을

보여줄수가 있다고 생각이 되겠습니다 성장을 하면서 상대방의 딜러들을 암살을

해줘야하는 챔피언이라고 생각이 되겠습니다 밑에서부터 공략을 시작하겠습니다

1.이블린이 패치가 되면서 많은 변화가 있었습니다 CC기도 추가가 되었습니다

은신의 경우 6레벨 이후에 은신이 발동이 되기 때문에 초반에 갱킹을 다니기에는

힘들다고 생각이 됩니다 상황에 따라서 가줄수도 있다고 생각이 되겠습니다 일단

매혹스킬이 있기 때문에 갱킹성공만 한다면 빠른 성장으로 다른 라인을 터트릴수

있는 챔피언이라고 생각이 됩니다 일단 데미지면에서 엄청나게 강력 하겠습니다

2.룬의 경우에는 지배 + 마법을 사용을 해주시면 되겠습니다

감전 : 3초동안 같은챔피언 공격3회나 스킬3회 공격시 추가피해가 들어갑니다

돌발일격 : 궁극기 + 벨트로 통해서 돌발일격을 활성화를 시켜줄수가 있기 때문

에 사용을 해주는 편이라고 봐주시면 될거 같습니다 시야수집이나 좀비와드를 

사용을 해주시면 되겠습니다 끈질긴 사냥꾼의 경우 빠르게 갱킹이 좋아집니다

기민함 또한 갱킹능력을 많이 올려주는 룬이라고 생각이 됩니다 기동력을 통해서

갱킹을 다니는 이블린에게 잘어울리는 룬이라고 생각 됩니다 저는 저렇게씁니다

3.이블린 아이템트리에 대해서 자세하게 알아보도록 하겠습니다

룬의 메아리 – 초기형 벨트 – 리치베인 – 리안드리의 고통 – 존야의 모래시계 입니다

상황 아이템 : 공허의 지팡이 + 라바돈의 죽음모자 + 밴시의 장막 정도가 되겠습니다

일단은 AP챔피언이기 때문에 AP아이템을 구축을 해주시는것이 좋다고 판단됩니다

벨트를 안올리고 아이템을 앞당기고 라바돈 OR 공허의 지팡이를 가주셔도 됩니다

일단은 아이템의 경우는 상황에 맞게 사용을 해주시는것이 중요하다고 생각되네요

4.신발의 경우에는 4가지를 선택을 해주시면 되는 부분입니다

기동력 : 이동속도가 많이 증가하기 때문에 갱킹능력이 매우좋아지는 신발 입니다

일단은 초반에 사용을 하다가 중후반에 판매를 하고 변경을 해주시면 되겠습니다

마법사의 신발 : 이블린의 경우 전체스킬이 AP데미지가 붙어있기 때문에 암살을

하기에 매우 좋아진다고 생각 되겠습니다 데미지가 매우강력해지는 신발입니다

닌자의 신발 : 상대방에 3AD 이상일 경우 빠르게 올려주시는것이 좋겠습니다

헤르메스 : 상대방 AP정글 + AP미드일경우 빠르게 올려주시는것을 추천합니다

5.스킬선마 순서의 경우에는 Q-W-E-R 또는 Q-E-W-R 순서대로 마스터 합니다

Q스킬 : 이블린이 원하는 방향으로 가시를 발사해 피해를 주게 되는 스킬입니다

재시전을 할경우 가까운 적에게 피해를 주는 주딜링 스킬이라 보면 되겠습니다

W선마 : 범위 증가 + 매혹시간 증가 + 마법저항력이 감소가 되기 때문에 강력

한 데미지를 넣어주실수가 있습니다 일단은 매혹시간 증가가 매우큰거 같네요

E선마 : 피해량 증가와 대상 최대 체력의 피해를 주기 때문에 강력한 스킬입니다

R스킬 : 이블린 궁극기의 경우 전방의 적에게 피해를 주고 사용을 한후에 뒤로

이동을 하게 되겠습니다 체력이 낮은 적일수록 데미지 증가하는 궁극기입니다

6.스펠의 경우에는 강타 + 점멸을 고정으로 사용을 해주시면 되겠습니다

일단은 유틸성이 좋은 점멸을 고정으로 사용을 해주는것이 좋다고 생각 되네요

W스킬을 걸고 점멸 E를 쓰면서 데미지를 넣어주실수가 있겠습니다 저의 경우

위에 스펠로 고정으로 사용을 해주는 편이 되겠습니다 일단은 도주시나 추격

을 할경우에도 용이하게 사용이 되는 스펠이기 때문에 고정으로 사용합니다
