미국에서 직수입한 하이드로탄 특수강화 마루코팅약으로 마루코팅으로 마무리

 일반코팅제 강도10배, 효율10배증가 NBA농구코트장 사용하는 코팅제로 내구성이 우수합니다.


 하이드로탄 ( Hydrothane / 수성 우레탄계 우드전용 코팅제 )

 특징

 우수한광택, 최고의 내구성을 자랑하는 우레탄계 우드전용 코팅제

 

 OHSA ( 미국 직업안전 위생국 ) 유해정보기준 29CFR1910.1200 승인된 안전한 코팅제

 

 자외선노출에의한 황변현상이 없는 특수강화 마루코팅제

 

강력한 피막형성으로 스크레지, 스커프, 힐마크, 오염에 강력한 실드을 자랑합니다. 

 

 탁월한 내구성으로 현존 최고의 코팅제입니다.
