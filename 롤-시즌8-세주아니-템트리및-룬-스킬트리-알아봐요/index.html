1줄 요약

  1. Q – 점멸을 통해서 순간적으로 파고들어서 에어본을 시켜줄수가 있겠습니다 …중략…  해주시면 되겠습니다 저의 경우에는 강타와 점멸을 고정으로 사용을 합니다

안녕하세요 오늘 롤 시즌8 세주아니 템트리및 룬 특성 스킬트리 대해 알려 드릴려고

합니다 세주아니는 다수의 군중 제어기를 보유한 전형적인 탱커형 챔피언이라고 보

면은 될거 같습니다 아군을 보호해줄수도 있고 방해를 해줄수 있는 챔피언 입니다

슬로우 스킬도 보유를 하고 있으면서 궁극기의 경우 한타기여도 높은 편인 챔피언

이라고 생각이 되겠습니다 상황에 따라서 기절 or 엄청난 슬로우 걸어주게 됩니다

1.일단은 에어본 + 슬로우 + 기절스킬을 보유하고 있게 되겠습니다

또한 기절당한 적이 얼어붙을 경우 첫스킬이나 기본공격을 할경우 체력비례의 딜을

하기 때문에 데미지도 강력한 편이라고 생각이 되겠습니다 일단은 앞에서 탱커역할

을 해주기에 매우적합한 챔피언이라고 생각이 되겠습니다 밑에서부터 공략에 대해

자세하게 설명을 드리도록 하겠습니다 룬의 경우 2가지를 선택을 해줄수 있겠습니다

2.룬의 경우에는 결의 + 정밀 또는 정밀 + 마법을 사용을 해주시면 되겠습니다

여진 : 적 챔피언 이동불가 상태로 만들면 방어력과 마법저항력이 증가합니다

이번에 룬이 바뀌면서 공격속도 9% + 체력 65가 증가해서 무난한 룬입니다

기민한 발놀림 : 공격속도 18% + 민첩함을 통해서 빠른 공속이 되겠습니다

기민함을 드는이유의 경우에는 패시브 스턴을 맞추기가 매우편해진다고 보

면 될거 같습니다 추노력이 매우강력해지기 때문에 잘어울리는 룬이 됩니다

또한 초반에 정글링 도움이 되는 룬이기도 합니다 체력회복 되기때문입니다

3.세주아니 아이템트리에 대해서 자세하게 알아보도록 하겠습니다

잿불거인 – 워모그의 갑옷 – 강철의 솔라리 – 가고일 돌갑옷 – 란두인의 예언 입니다

상황 아이템 : 적응형 투구 + 가시갑옷 + 얼어붙은 심장 + 기사 맹세 + 망자의 갑옷

+ 얼어붙은 건틀릿 + 심연의 가면 정도로 봐주시면 될거 같습니다 무난하겠습니다

일단 잿불거인과 워모그가 나올 경우 유지력이 매우좋아지는 부분입니다 또한 세주

아니 패시브로 통해서 방어력과 마법저항력이 증가하기 때문에 어느정도 탱킹능력

을 보여줄수가 있게 되겠습니다 한타임에서 매우강력한 모습을 보여주기 쉽습니다

4.신발의 경우에는 3가지를 선택을 해주시면 되는 부분이 되겠습니다

닌자의 신발 : 상대방 AD + 원거리딜러 + 평타딜러들이 잘컸을 경우 가줍니다

헤르메스 : 상대방AP정글 + AP미드 + CC기가 많을 경우에 올려주면 되겠습니다

기동력 : 빠른 갱킹을 다닐수가 있겠습니다 갱킹능력도 강력한 편에 속한다고 생

각이 되겠습니다 일단은 스킬 CC기가 많기 때문에 효율이 매우좋은 편이 됩니다

신발의 경우 3가지만 사용을 하는 편이 되겠으며 기동력의 경우에는 초반에 사고

중후반에 판매를 하고 방어아이템 신발을 사용을 해주시는것이 중요하겠습니다

5.세주아니 스킬트리의 경우에는 W-E-Q-R 순서대로 마스터를 해주시면 되겠습니다

Q스킬 : 쿨타임 감소 + 적들에게 들어가는 데미지 증가하겠습니다 에어본용 입니다

W선마 : 쿨타임 감소 + 데미지증가 입니다 일단 2대를 치기 때문에 강력한 데미지

를 보여주기도 하고 정글링에 있어 광역데미지 들어가기 때문에 W선마를 해줍니다

E스킬 : 서리 중첩 쌓인적에게 피해량 증가 + 기절시간이 증가하게 되겠습니다 일단

기절시간 많이 증가하기 때문에 효율이 좋은 스킬이라고 생각 해주시면 되겠습니다

6.스펠의 경우에는 강타 + 점멸을 고정으로 사용을 해주시면 되겠습니다

Q – 점멸을 통해서 순간적으로 파고들어서 에어본을 시켜줄수가 있겠습니다

또한 들만한게 없겠습니다 정글이기 때문에 강타 + 점멸을 고정으로 사용을 

해주시면 되겠습니다 저의 경우에는 강타와 점멸을 고정으로 사용을 합니다
