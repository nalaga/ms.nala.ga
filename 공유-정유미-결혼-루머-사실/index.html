공유 정유미 결혼 루머 사실?

  • By admin
  • on 1월 13, 2018
  • in 미분류

3줄 요약

  1. 왜 그럼 이런 루머가 뜨는 걸까요
  2. 소속사에서 친해졌나보네요. (또한 공유 정유미 고향이 모두 부산이라서 더 …중략… 속사는 공유 정유미 결혼에 대한 루머가 사실무근이라고 오늘도 밝혔습니다.
  3. 도가니에 함께 출연을 하면서 알게되었습니다.

안녕하세요.

작년 10월에 뜬 공유 정유미 결혼에 대한 루머가 오늘도 다시 언급이 되고 있네요.

오늘뜬 루머는 인터넷 커뮤니티에서 나왔는데요.

상당히 구체적이었습니다.

공유 정유미가 결혼을 어디에서 하고, 언제하는지에 대한 날짜와 장소도 언급이 되었다고 하니 말입니다.

왜 그럼 이런 루머가 뜨는 걸까요

아마도 공유 정유미 소속사가 같아서 그런것 같은 느낌인데요.

둘다 모두 매니지먼트 숲에 소속이 되어 있습니다.

2016년에 정유미가 매니지먼트 숲으로 옮기게 되었고, 공유는 소속이 된 상태여서 둘은 같은 소속사에서 일을 하게 되었습니다.

벌써 같은 소속사에서 일한지 3년이나 되네요.

소속사에서 친해졌나보네요. (또한 공유 정유미 고향이 모두 부산이라서 더 친해졌을수도 있을 것 같습니다.)

하지만 소속사는 공유 정유미 결혼에 대한 루머가 사실무근이라고 오늘도 밝혔습니다.

사실 둘은 2011년 영화 도가니에서 만났는데요.

도가니에 함께 출연을 하면서 알게되었습니다.

그리고 연애조작단 시라노라는 드라마에 정유미가 출연중이었는데, 공유가 깜짝 출연을 하기도 하였습니다.

마지막으로 같이한 작품은 바로 부산행이라는 영화입니다.

한국에서 꽤나 대박을 터트린 영화이기에 많이들 아시겠죠.

천만관객을 동원한 영화이니 말입니다. 

진실은 과연 도대체 어디에 있는것인지 궁금하기도하네요.

일단 공유 정유미 나이 차이는 4살인데요.

올해 40살이 되는 공유와 올해 36살인 정유미는 나이차이로만 보면 천생연분이네요.

4살차이니 말입니다.

그래도 아직은 커플이 아니고 결혼에 대한 이야기가 사실무근이라고 하니 소속사 이야기를 믿어봐야겠지요. 

사실이면 곧 소식이 나오지 않을까 싶습니다.

그럼 공유 정유미 결혼 소식 루머를 기념으로 프로필 한번 알고 마치도록 하겠습니다. 

공유 프로필은 아래와 같습니다.

공유 본명 : 공지철 / 나이 : 1979년 7월 10일 (40세) / 고향 : 부산 / 키 : 184cm / 몸무게 : 74kg / 혈액형 : A형

소속사 : 매니지먼트 숲 / 학력 : 경희대학교 

정유미 프로필은 아래와 같습니다. 

나이 : 1983년 1월 18일 (36세) / 키 : 163cm / 소속사 : 매니지먼트 숲 / 고향 : 부산

학력 : 서울예술대학교 영화과

그럼 이만 줄일께요.
